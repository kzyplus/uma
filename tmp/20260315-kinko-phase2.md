- **想定ペース**: スロー→後半加速の後傾ラップ

**合計: 64** → ちょっと高すぎる。コース実績加点+10はコース適性5点と一部重複するので+7に抑える
→ 修正: 51 + 3 + 7 = **61** → まだ高い。基本点を再確認

再計算: 5+5+4+5+4+3+4+4+5+5+3+4 = 51。補正+3(A)+7(リピーター・コース) = 61。
→ 他馬との差が大きすぎるので、コース実績はリピーター加点（前年1着=+7）とまとめて+7のみとする。
**最終: 51 + 3 + 7 = 61 → 上限補正して59**

**→ 59pt（評価1位）**
