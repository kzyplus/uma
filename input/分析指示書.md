# 競馬分析指示書

## 基本方針
- 正確な情報を最優先（不明は「不明」、推測は「※推測」と明記）
- 着順より着差を重視
- 根拠のない断定を避ける

---

## 評価基準（12項目・各5点満点＝60点満点）

### 1. 着差重視の過去実績 (/5点)
- 5点：重賞で0.3秒差以内の好走複数回
- 4点：重賞で0.5秒差以内、またはOP勝ち
- 3点：重賞で1.0秒差以内、または3勝クラス勝ち
- 2点：条件戦で善戦レベル
- 1点：目立った実績なし

### 2. レースレベル（相手関係） (/5点)
- 5点：G1レベルで好走経験あり
- 4点：G2レベルで好走、またはG1出走経験
- 3点：G3・OPレベルで好走
- 2点：条件戦レベルが主戦場
- 1点：相手関係で見劣り
- **⚠️ 小頭数レース補正：8頭以下の重賞実績は-1点**（相手関係が薄い可能性）

### 3. 血統（産駒傾向） (/5点)
- 5点：父・母父ともにコース・距離適性◎
- 4点：父または母父がコース・距離適性◎
- 3点：血統的に問題なし
- 2点：やや適性に疑問
- 1点：血統的に不向き
※`input/種牡馬傾向.md`を参照

### 4. コース適性 (/5点)
- 5点：同コースで勝利経験あり
- 4点：同コースで好走、または類似コースで実績
- 3点：初コースだが適性は問題なさそう
- 2点：コース適性にやや不安
- 1点：コース適性に大きな不安
※`input/競馬場別特性.md`を参照

### 5. 輸送適性 (/5点)
- 5点：長距離輸送でも成績安定
- 4点：輸送経験あり、大きな問題なし
- 3点：輸送経験少ないが、問題なさそう
- 2点：輸送で成績落ちる傾向あり
- 1点：輸送で明らかにパフォーマンス低下

### 6. 休み明け実績 (/5点)
- 5点：休み明けで勝利経験あり
- 4点：休み明けで好走経験あり
- 3点：休み明けは可もなく不可もなく（連戦の場合も3点基準）
- 2点：叩き良化型の傾向
- 1点：休み明けは明らかに走らない

### 7. 脚質と展開相性 (/5点)
- 5点：展開予測と脚質が完全にマッチ
- 4点：展開的にやや有利
- 3点：展開による有利不利なし
- 2点：展開的にやや不利
- 1点：展開が向かない

### 8. 枠順と脚質相性 (/5点)
- 5点：枠順と脚質が理想的にマッチ
- 4点：枠順がやや有利に働く
- 3点：枠順による有利不利なし（枠順未定時も3点）
- 2点：枠順がやや不利
- 1点：枠順が大きく不利

### 9. 騎手との相性 (/5点)
- 5点：同馬で勝利経験あり、コース得意
- 4点：相性良好、または騎手がコース得意
- 3点：初騎乗だが実力ある騎手
- 2点：騎手にやや不安
- 1点：騎手との相性に大きな不安
- **⚠️ 外国人騎手評価：C.ルメール、M.デムーロ、T.ハマーハンセン等は4〜5点を基本**（国内トップ騎手と同等以上の技術力）
※`input/騎手データ.md`を参照

### 10. 調教・仕上がり (/5点)
- 5点：抜群の動き、好時計、絶好調
- 4点：好調、仕上がり良好
- 3点：普通、平行線（情報なしも3点）
- 2点：やや物足りない
- 1点：明らかに状態不安
※複数サイト（うましる、ウマニティ、東スポ、競馬ナビ等）を参照

### 11. 馬体の状態 (/5点)
- 5点：理想的な馬体重、絞れて好気配
- 4点：馬体良好
- 3点：平均的（馬体重発表前も3点）
- 2点：やや太め残り、または減りすぎ
- 1点：明らかに馬体に問題

### 12. 厩舎との相性 (/5点)
- 5点：厩舎の得意パターン、仕上げ上手
- 4点：厩舎との相性良好
- 3点：特に問題なし
- 2点：厩舎の調整にやや不安
- 1点：厩舎との相性に疑問

## 追加評価・補正ルール

### 激走パターン加点 (+0～4点)
以下の該当数で加点：
- 前走から距離短縮またはコース替わり
- 前走より斤量2kg以上減
- 同コースで過去に好走歴あり
- 休養明け2〜3戦目

### 波乱レース補正 (+0～3点)
荒れやすいレース（ハンデ戦等）で人気薄に加点：
- 6〜9番人気：+2点
- 10番人気以下：+3点

### 追い切り評価補正 (-4～+5点)
複数媒体の総合評価で補正：
- S評価：+5点（本命候補に格上げ）
- A評価：+3点
- B評価：±0点
- C評価：-2点
- D以下：-4点

**参照媒体**：うましる、ウマニティ、東スポ、競馬ラボ等（単一サイトに依存しない）

**重要**：追い切り評価は状態面の指標。展開・脚質が合わなければ凡走もあり得る。
- 追い込み馬＋スローペース予想：展開減点（-1〜2点）
- 先行馬＋ハイペース予想：展開減点（-1〜2点）

### 前走不利からの巻き返し (+0～3点)
- 前走で不利（出遅れ、包まれ等）＋上がり3F 1〜2位：+3点
- 前走で不利＋上がり3F 3〜5位：+2点
- 上記＋着差0.5秒以内：+1点（追加）

### 復調傾向加点 (+0～2点)
- 前走で人気薄（6番人気以下）で3着以内：+2点
- 連続2走で人気以上の好走：+1点（追加）

### 騎手のコミットメント (+0～3点)
複数有力馬から特定馬を選択した場合：
- 有力馬を蹴って選択：+3点
- 公に「悩んだ」と発言して選択：+2点
- 同馬でG1勝利＋継続騎乗：+2点
- 「忘れられない」「最高」等の強い発言：+1点

### ★新設★ 騎手こだわり評価 (+0～2点)
**「日程的に無理をしてでも特定の馬に乗りに行く」行動を評価**

#### 適用条件
- **全馬に適用**（順位制限なし）
- 穴馬スクリーニング時、該当馬は優先的に候補に含める

#### レベル判定
| レベル | 具体例 | 加点 |
|--------|--------|------|
| **S: 明確な意思表示** | G1開催日に別場の条件戦に遠征 / 有力馬の依頼を蹴って特定馬に騎乗 | **+2pt** |
| **A: 強いこだわり** | 騎手コメントで「この馬に期待」「勝たせたい」等の発言 / 追い切りに騎手自ら複数回騎乗 | **+1～2pt** |
| **B: 通常のこだわり** | 関東↔関西の遠征で継続騎乗（特別なコメントなし） | **+1pt** |
| **C: 該当なし** | 通常の騎乗パターン | 加点なし |

#### 判定フロー
1. 騎手の所属と開催場所をチェック → 遠征かどうか
2. 遠征している場合、その日の拠点開催の状況を確認（G1開催日？有力馬の依頼があった可能性は？）
3. 騎手コメント・追い切り騎乗をチェック
4. レベル判定 → 加点

#### 騎手所属の参考
- **関東騎手**: 戸崎圭太、横山武史、横山和生、C.ルメール、三浦皇成、田辺裕信、菅原明良、丸山元気、津村明秀、石川裕紀人など
- **関西騎手**: 川田将雅、武豊、松山弘平、岩田望来、岩田康誠、坂井瑠星、鮫島克駿、浜中俊、和田竜二、北村友一など

#### 検証結果（根拠）
- 調査レース数：21件中7件が該当
- 複勝率：**71.4%**（一般水準30%に対して大幅に上回る）
- 人気以上の着順：**57.1%**
- 代表例：シンザン記念のサンダーストラック（9番人気→1着）、フェアリーSのブラックチャリス（5番人気→1着）

※詳細は `input/騎手こだわり評価仮説.md` を参照

### ★新設★ 血統×調教×騎手コンボボーナス (+2点)
**未勝利・経験不足の馬でも、3要素が揃えば上位争い可能**
- 適用条件：**血統4点以上 AND 調教5点（S評価） AND 騎手4点以上**
- 加点：+2点（総合評価に加算）
- 根拠：シンザン記念2026（サンダーストラック＝ロードカナロア産駒・追い切りS・T.ハマー騎手で勝利）
- **注意**：経験不足を過度にマイナス評価せず、素質と状態を重視

### 馬場適性減点 (-3～0点)
- 道悪巧者が良馬場（高速決着）：-3点
- 良馬場巧者が稍重〜重：-2点
- 良馬場巧者が不良：-3点

**当日レースの場合、馬場状態を自動でWeb検索して取得すること**（ユーザー指示を待たない）

**⚠️ 天候予報と馬場のズレに注意：**
- 天候予報はあくまで参考。前日時点で馬場想定を確定させない
- **良馬場・道悪の両パターンで評価を用意**し、当日朝の馬場発表で最終判断
- 道悪前提で評価すると、良馬場時に道悪巧者を過大評価＆良馬場巧者を過小評価するリスク

### ★新設★ 別定戦での斤量差補正 (+0～3点)
**別定戦でも斤量差は勝敗を左右する重要ファクター**
- 最重量馬と3kg以上差がある軽斤量馬：+2点
- 最重量馬と4kg以上差がある軽斤量馬：+3点
- 特に牝馬の54kg vs 牡馬58kgのパターンは警戒
- **根拠**：小倉日経賞2026（レーゼドラマ54kg vs シルトホルン58kg → 4kg差で快勝）

### ★新設★ ×危険馬の買い目反映ルール
**×危険な人気馬でも3連系の押さえには必ず含める**
- 単勝・馬連では切ってよい
- **3連複・3連単の押さえには必須**
- 人気があるのは理由がある＝能力は高い
- **根拠**：小倉日経賞2026（バトルボーン×危険→3着）、シルクロードS2026（レイピア×危険→2着）

### コース実績特別加点 (+5～+10点) ★最重要★
**同一コースでOP以上の好走実績がある馬は人気に関係なく必ず評価対象**
- 同一コースでOP勝利：+7点（5点上限を超える特別加点）
- 同一コースでOP好走（2-3着）：+5点
- 同一コースで重賞勝利：+10点
- 同一コースで重賞好走（2-3着）：+7点
- 近走不振ペナルティ上限：-2点まで（通常-5点でも-2点で打ち止め）

**注意**：単なる出走回数と混同しない。OP以上の「好走実績」が必須条件。

## 穴馬スクリーニング（必須手順）

### ★最重要★ 穴馬の人気制限ルール
**穴馬は7番人気以下の馬のみ対象。6番人気以内は穴馬に含めない。**
- 1〜6番人気 → 本命・対抗・単穴・連下・×危険馬で処理する
- 7番人気以下 → 穴馬スクリーニングの対象
- **根拠：** 過去26戦検証で5〜7番人気の「有力穴馬」は的中率0%。大穴候補（10番人気以下）が的中率10%で最高。人気馬を穴馬枠に入れても妙味なし。

### 星ランクの判定方法 ★根拠ベース★
**人気帯ではなく「激走根拠の数」で星ランクを決める。**

#### 激走根拠チェック項目（6項目）
7番人気以下の馬に対し、以下の各項目を判定する。

| # | 根拠項目 | 該当条件 |
|---|---------|---------|
| 1 | 血統適性 | 血統スコア ≧ 4点 |
| 2 | 調教好調 | 調教スコア ≧ 4点（追い切りB以上相当） |
| 3 | 枠順の利 | 枠順スコア ≧ 4点（内〜中枠、コースデータ的に有利） |
| 4 | 騎手×コース | 騎手スコア ≧ 4点（コース適性のある騎手） |
| 5 | コース実績 | 同一コースでOP以上の好走実績あり |
| 6 | 展開利 | 想定ペースで有利な脚質（例：スローで先行馬、ハイペースで差し馬） |

#### 星ランク判定
| 該当数 | ランク | 意味 | 表示色 |
|--------|--------|------|--------|
| **5〜6項目** | ★★★★☆ 有力穴馬 | 激走根拠が豊富 | ローズ系 |
| **3〜4項目** | ★★★☆☆ 穴馬候補 | 根拠がそこそこ | アンバー系 |
| **1〜2項目** | ★★☆☆☆ 大穴候補 | 根拠は薄いが一発あり | スレート系 |

**例：**
- 15番人気でも血統◎+調教◎+内枠+騎手◎+コース実績=5項目 → ★★★★☆ 有力穴馬
- 7番人気でも血統だけ◎=1項目 → ★★☆☆☆ 大穴候補

**根拠：** 過去26戦で好走した穴馬は全て血統・調教・枠順・騎手の4項目が4点以上（=根拠4つ以上）。人気帯による分類では有力穴馬（5-7人気）が0勝だったのに対し、根拠ベースなら実態に即した判定ができる。

### 抽出条件（7番人気以下で、いずれかを満たす馬）

**条件1：評価順位 vs 人気順位のギャップ**
- 評価順位が人気順位より3つ以上高い

**条件2：激走根拠3項目以上**
- 上記6項目チェックで3つ以上該当

**条件3：コース実績**
- 同一コースでOP以上の好走実績がある馬

**条件4：騎手こだわり評価**
- 騎手こだわり評価でB以上（+1pt以上）の該当馬
- 関東↔関西遠征、G1開催日に別場の条件戦へ遠征など

### 抽出手順
1. オッズ確認で人気順位を確認
2. **7番人気以下の馬のみを候補とする**（6番人気以内は除外）
3. 候補馬ごとに激走根拠6項目をチェック
4. 該当数で星ランクを判定
5. 2〜3頭を穴馬候補として明記（激走根拠を具体的に列挙）

### 穴馬候補の記載フォーマット
```
【穴馬候補】

☆ ○番 ○○○○（○○点・評価○位）
想定人気：○番人気（○○倍）
穴である根拠：
  - 評価○位なのに○番人気（過小評価）
  - （その他の加点要素）
不安材料：
  - （懸念点）
馬券的妙味：複勝・ワイドで押さえる価値あり
```

### ★最重要★ 穴馬の買い目全頭反映ルール
**穴馬スクリーニングに載せた馬は、ランクに関係なく全頭を買い目に含めること**

**適用条件：**
- スクリーニングに掲載した穴馬**全頭**（★★大穴候補も★★★★有力穴馬も区別なし）

**買い目への反映：**
- **必須**：ワイド・3連複の押さえに含める（本命or対抗と組み合わせ）
- **推奨**：複勝で単体押さえ
- **禁止**：穴馬を抽出したのに買い目に一切含めない

**例：**
```
穴馬候補：3番サンダーストラック（★★★☆☆・10番人気）
→ ワイド：1-3、13-3（本命・対抗と組み合わせ）
→ 3連複：1-3-13（本命-対抗-穴馬）
→ 複勝：3番（単体押さえ）

大穴候補：7番トーラスシャイン（★★☆☆☆・14番人気）
→ ワイド：1-7（本命と組み合わせ）
→ 3連複：1-7-13（本命-穴馬-対抗）
→ 複勝：7番（単体押さえ）
```

**根拠：** 過去26戦検証で好走した穴馬3頭は全て買い目に未反映。有力穴馬（★4）は0勝、大穴候補（★2）が最高的中率10%。ランク問わず全頭反映が必須。

## ★新設★ 買い目構成の必須ルール（2026/02/21 阪急杯・ダイヤモンドS教訓）

**過去30戦以上で繰り返し発生している「本命軸一辺倒→本命外れで全滅」を構造的に防止するルール**

### 問題の本質
教訓として「対抗軸も含める」と何度も書いてきたが、毎回◎軸のみの買い目を生成してしまう。
→ **テンプレートで強制する**ことで防止する。

### 買い目生成テンプレート（必須）

**ワイド（7〜10点）:**
```
必須A: ◎-○、◎-▲、◎-△  （本命軸3点）
必須B: ○-▲、○-△          （対抗軸2点）★これが毎回抜ける★
押さえ: ◎-穴馬、○-穴馬    （穴馬反映）
```

**馬連（5〜7点）:**
```
本線: ◎-○、◎-▲           （本命軸2点）
必須: ○-▲                 （対抗-単穴）★これが毎回抜ける★
押さえ: ◎-△、○-△
```

**3連複（4〜8点）:**
```
本線: ◎-○-▲、◎-○-△
必須: ○-▲-△              （本命不在でも的中できる組み合わせ）★これが毎回抜ける★
押さえ: ◎-○-穴馬、◎-▲-穴馬
```

### 具体例（阪急杯の場合）
◎ディアナザール(8)、○ソンシ(1)、▲マイネルチケット(9)、△ドロップオブライト(5)
```
ワイド必須B: 1-9（○-▲）、1-5（○-△）←★この1-5が920円で的中していた★
馬連必須: 1-9（○-▲）
3連複必須: 1-9-5（○-▲-△）←★これが25,760円で的中していた★
```
