# 競馬分析指示書

## 分析の基本方針

### 最重要事項
- **正確な情報を最優先**する
- 不明な情報は「不明」と明記し、推測は「※推測」と明示する
- 根拠のない断定は避ける

## 分析時の注意事項

### やるべきこと
- 登録済みデータ（競馬場、種牡馬、騎手）を必ず参照
- 馬場状態別（良・道悪）の評価を必ず記載
- 不安材料は正直に記載
- 過去の対戦結果があれば言及

### やってはいけないこと
- 根拠なく「絶対」「確実」などの表現を使う
- 人気だけで評価を決める
- 調べられる情報を調べずに推測で埋める

## 追加分析ルール（JC2025反省より）

### 【過去データの扱い方】

#### 原則
- 過去データは「傾向」であり「法則」ではない
- 10年以上続いた傾向でも、破られる可能性は常にある
- データより**個体の絶対能力**を優先する

#### 具体的な運用
| 状況                                 | 対応                                                   |
| ------------------------------------ | ------------------------------------------------------ |
| 過去データと矛盾する有力馬がいる場合 | 「データ上は不利だが、能力で覆す可能性あり」と両論併記 |
| 「○○は過去10年で好走なし」系のデータ | 参考情報に留め、評価を大幅に下げる根拠にしない         |

---

### 【外国馬の評価基準】

#### 従来の問題点
- 「外国馬は日本で苦戦」という過去傾向を過信
- 欧州馬＝パワー型という固定観念

#### 新しい評価基準
1. **世界ランキング・レーティング**を重視
   - 世界ランク5位以内 → 日本馬場適応の可能性を高く見積もる

2. **上がり3F実績**を確認
   - 欧州でも上がり33秒台を使えている馬 → 瞬発力あり、日本向き

3. **来日後の調整状況**を重視
   - 陣営の遠征実績
   - 来日後の調教内容・動き

4. **直接対決の結果**
   - 日本馬との対戦があれば、その着差を重視
   - ただし馬場・展開の違いを考慮

#### 評価補正
| 条件                     | 補正 |
| ------------------------ | ---- |
| 世界ランク1～3位         | +3点 |
| 世界ランク4～10位        | +1点 |
| G1・3勝以上              | +2点 |
| 来日後調教良好           | +1点 |
| 過去に日本で好走経験あり | +2点 |

---

### 【上がり3F分析の強化】

#### 東京競馬場（特に2400m）
- **最重要指標**として上がり3Fを評価
- 過去レースで上がり最速を記録した回数をカウント

#### 評価基準
| 上がり実績               | 評価 |
| ------------------------ | ---- |
| G1で上がり最速経験あり   | +2点 |
| 重賞で上がり33秒台複数回 | +1点 |
| 上がり34秒台が多い       | ±0点 |
| 上がり35秒以上が多い     | -1点 |

#### 分析時の必須項目
- 直近5走の上がり3Fを列挙
- 上がり順位（何番目に速かったか）を記載

---

### 【枠順評価の相対化】

#### 原則
- 枠順は「有利・不利」程度の評価に留める
- 枠順だけで大幅な減点をしない

#### 具体的な運用
| 従来                                | 改善後                                                  |
| ----------------------------------- | ------------------------------------------------------- |
| 「8枠は過去10年連対なし」→ 大幅減点 | 「8枠は不利だが、能力で克服可能」→ 小幅減点（-1点程度） |
| 「1枠は好走率高い」→ 大幅加点       | 「1枠は有利」→ 小幅加点（+1点程度）                     |

---

### 【ペース予測と展開分析】

#### 逃げ馬の評価
- 逃げ馬が**単騎で行けるか**を重視
- 逃げ馬が複数いる場合 → 差し有利と予測

#### 展開予測の両論併記
必ず以下の2パターンを想定：
1. **スロー想定**：有利な馬は？
2. **ハイペース想定**：有利な馬は？

---

### 【「危険な人気馬」認定の厳格化】

#### 認定条件（以下の複数を満たす場合のみ）
- [ ] 過去の同条件で凡走歴あり
- [ ] 直近のパフォーマンスが下降傾向
- [ ] 明確な不安材料（怪我明け、距離不安など）
- [ ] 枠順・展開が明らかに不利

#### 認定してはいけないケース
- 「過去データ上不利」だけが根拠
- 実績十分で、直近も好調な馬
- 世界トップクラスの実力馬

---

### 【評価の両論併記ルール】

上位人気馬（1～5番人気想定）については、必ず以下を記載：
```
【プラス材料】
・
・

【マイナス材料】
・
・

【結論】
本命 or 対抗 or 軽視（理由を明記）

【覆る可能性】
軽視評価の場合、どんな条件なら好走するかを記載
```

### 【最終確認チェックリスト】

分析完了後、以下を確認：

- [ ] 過去データだけで大幅減点した馬はいないか？
- [ ] 外国馬を「傾向」だけで軽視していないか？
- [ ] 上がり3F性能を十分に評価したか？
- [ ] 展開予測は複数パターン想定したか？
- [ ] 「危険な人気馬」の根拠は十分か？
- [ ] 能力上位馬を枠順だけで軽視していないか？
- [ ] 穴馬スクリーニングを実施したか？

## 必須手順

### 穴馬スクリーニング
総合評価完了後、以下の手順で穴馬候補を抽出する：

#### 【抽出条件】（以下のいずれかを満たす馬）

**条件1：評価順位 vs 人気順位のギャップ（有馬記念2025で有効性確認済み）**

| 条件 | 判定 |
|------|------|
| 評価順位が人気順位より**3つ以上高い** | **穴馬候補** |
| 評価順位が人気順位より**5つ以上高い** | **有力穴馬候補** |

**具体例：有馬記念2025**

| 馬名 | 評価 | 人気 | ギャップ | 結果 |
|------|------|------|----------|------|
| コスモキュランダ | 6位 | 11番人気 | **+5** | **2着** |
| タスティエーラ | 4位 | 8番人気 | **+4** | 5着 |
| マイネルエンペラー | 9位 | 12番人気 | **+3** | 14着 |

**条件2：激走パターン（従来ルール）**
- 「激走パターン」が2点以上の馬

#### 【抽出手順】

1. **オッズ確認**: 前日または当日オッズで人気順位を確認
2. **ギャップ計算**: 評価順位 - 人気順位 を算出
3. **抽出**: ギャップ+3以上、または激走パターン2点以上の馬
4. **記載**: 2〜3頭を【穴馬候補】として明記

#### 穴馬候補の記載フォーマット
```
【穴馬候補】

☆ ○番 ○○○○（○○点・評価○位）
想定人気：○番人気（○○倍）
穴である根拠：
  - 評価○位なのに○番人気（過小評価）
  - （その他の加点要素）
不安材料：
  - （懸念点）
馬券的妙味：複勝・ワイドで押さえる価値あり
```

## 2歳G1特別ルール（ホープフルS2025反省より追加）

### 【キャリア1〜2戦馬の評価】

#### 原則
- キャリアの少なさだけで大幅減点しない
- 「実績不足」ではなく「ポテンシャル評価」に切り替える
- 過去にキャリア1戦でG1制覇した例：リオンディーズ（2015年朝日杯FS）、ロブチェン（2025年ホープフルS）

#### 新馬戦内容評価（/5点）※2歳限定戦専用
| 点数 | 基準 |
|------|------|
| 5点 | 着差3馬身以上 or 上がり最速で圧勝 |
| 4点 | 着差2馬身以上 or 余裕を持った勝ち方 |
| 3点 | 着差1馬身程度の普通の勝ち方 |
| 2点 | 辛勝（クビ・ハナ差） |
| 1点 | 新馬戦未勝利 or 負けて未勝利勝ち上がり |

#### 追加評価ポイント
- [ ] 脚質の幅を見せた（逃げ→差し、または差し→逃げに変化対応）：+1点
- [ ] 不利（出遅れ、包まれ等）を跳ね返して勝利：+1点
- [ ] 上がり最速を記録：+1点
- [ ] 2戦目で着差拡大（成長力）：+1点

### 【新種牡馬の評価基準】

#### 原則
- 「データなし＝評価できない」ではなく「父馬の実績から推測」
- 種牡馬傾向.mdにない場合でも、以下の手順で評価

#### 評価手順

**Step 1: 父馬自身のG1勝ちコースを確認**
| 父馬のG1勝利コース | 産駒の推測適性 |
|-------------------|---------------|
| 中山で勝利経験あり | 中山◎ |
| 東京で勝利経験あり | 東京◎ |
| 長距離G1勝利 | スタミナ型◎ |
| スプリントG1勝利 | スピード型◎ |

**Step 2: 父馬の血統系統を確認**
- サンデーサイレンス系 → 瞬発力寄り
- ノーザンダンサー系 → パワー・スタミナ寄り
- ミスタープロスペクター系 → 万能〜ダート寄り

**Step 3: 初年度産駒の傾向（あれば）**
- 他の産駒がいれば、その成績から傾向を推測

#### 具体例：ワールドプレミア（2025年ホープフルS勝ち馬の父）
| 父馬の競走成績 | 推測される産駒傾向 |
|---------------|-------------------|
| 菊花賞1着（京都3000m） | スタミナ◎ |
| 天皇賞春1着（阪神3200m） | 長距離◎ |
| 有馬記念1着（中山2500m） | **中山◎** |
| 父ディープインパクト | 瞬発力の可能性あり |

→ **結論：中山適性は父自身の実績から推測可能だった**

### 【コース替わりリスク評価】

#### 原則
- 前走と今回のコース特性の違いを必ず確認
- 特に「東京→中山」「京都→中山」は要注意

#### コース特性比較表
| コース | 直線 | 坂 | 特徴 |
|--------|------|-----|------|
| 東京 | 525m（長） | 緩 | 瞬発力勝負、差し有利 |
| 中山 | 310m（短） | 急 | パワー・先行有利 |
| 京都 | 404m（中） | 3角坂 | 下り利用の仕掛け |
| 阪神外 | 474m（長） | 有 | 東京に近い |

#### コース替わりリスク評価（/5点）
| 点数 | 基準 |
|------|------|
| 5点 | 同コースで実績あり |
| 4点 | 類似コース（直線長さ・坂の有無が近い） |
| 3点 | 初コースだが血統的に問題なし |
| 2点 | **真逆のコース特性への変更**（東京→中山など） |
| 1点 | コース適性に明確な疑問 |

#### 要注意パターン
| 前走 | 今回 | リスク |
|------|------|--------|
| 東京（直線長・坂緩） | 中山（直線短・急坂） | **高** |
| 新潟外（直線最長・平坦） | 中山 | **高** |
| 札幌（洋芝・平坦） | 中山（野芝・急坂） | **中〜高** |
| 京都 | 中山 | 中 |
| 阪神内 | 中山 | 低 |

## 追い切り評価の重視（有馬記念2024反省より追加）

### 【基本原則】

追い切り評価は**近走成績より優先**する場合がある。特に以下のケースでは追い切り評価を最重視する：

| ケース | 対応 |
|--------|------|
| 近走不振だが追い切りA以上 | **復調の可能性、軽視禁止** |
| 人気馬だが追い切りB以下 | **危険な人気馬候補** |
| 人気薄だが追い切りA以上 | **穴馬候補に浮上** |

### 【追い切り評価による点数補正】

| 追い切り評価 | 補正 | 対応 |
|-------------|------|------|
| S評価 | **+5点** | 本命候補に格上げ |
| A評価 | **+3点** | 評価維持〜格上げ |
| B評価 | ±0点 | 現状維持 |
| C評価 | -2点 | やや格下げ |
| D以下 | **-4点** | 危険視・大幅格下げ |

### 【追い切り評価の確認項目】

分析時に以下を必ず確認：

- [ ] 1週前追い切りの内容・時計
- [ ] 最終追い切りの内容・時計
- [ ] 1週前→最終での変化（良化 or 下降）
- [ ] 併せ馬の結果（先着・併入・遅れ）
- [ ] 陣営コメント（状態に関する言及）

### 【追い切り評価の情報源】

以下のサイトで追い切り評価を確認：

1. **うましる** - S〜E の6段階評価
2. **東スポ競馬** - 調教採点・調教班の鬼ジャッジ
3. **ウマニティ** - 調教診断（水曜・最終版）
4. **アギョウトレセン** - 追い切り評価
5. **netkeiba** - 調教映像・調教タイム

### 【追い切り評価の具体例：有馬記念2024】

| 馬名 | 近走 | 追い切り | 総合判断 | 結果 |

| シャフリヤール | 凡走続き | **A** | 穴馬候補に浮上 | **2着** |

### 【追い切り評価と近走成績の優先順位】

| 状況 | 優先すべき情報 |

## 前走不利からの巻き返し評価（マイルCS2025反省より追加）

### 【基本原則】

前走で不利があった馬でも、上がり3Fが優秀であれば次走での巻き返しが期待できる。
着順だけで判断せず、**レース内容**を精査することが重要。

### 【確認すべき不利の種類】

| 不利の種類 | 内容 |
|------------|------|
| 出遅れ | スタートで後手、ゲート内で立ち上がるなど |
| 包まれ | 馬群に閉じ込められて進路がなかった |
| 進路カット | 前の馬が斜行して進路を失った |
| 接触・躓き | 他馬との接触、躓いてバランスを崩す |
| 馬場の不利 | 荒れた内を通らされた、重馬場で脚を取られた |
| 外を回す | 終始外を回されてロスが大きかった |

### 【巻き返し評価の点数補正】

| 条件 | 補正 |
|------|------|
| 前走で不利あり＋上がり3F **1〜2位** | **+3点**（巻き返し候補） |
| 前走で不利あり＋上がり3F **3〜5位** | **+2点** |
| 前走で不利あり＋着差0.5秒以内 | **+1点**（追加加点） |
| 上記＋今回追い切りA以上 | **穴馬候補に浮上** |

### 【具体例】

**レガレイラ（有馬記念2024・1着）**
- 前走：エリザベス女王杯4着
- 不利：終始外を回すロス
- 上がり3F：メンバー中2位（33.8秒）
- 追い切り：A〜A+

→ 「不利＋上がり上位＋追い切り良好」で**巻き返し候補**

**ウォーターリヒト（マイルCS2025・3着/15番人気）**
- 前走：富士S（馬場状態に泣いた）
- 上がり3F：メンバー中最速クラス
- コメント：「上がり最速で突っ込んでくる瞬発力」

→ 前走不利ルールで拾えた可能性あり

## 厩舎別調教スタイルの評価（追い切り評価の補正）

### 【基本原則】

追い切り時計は厩舎によって評価を変える必要がある。
**馬なり主体の厩舎**は時計が出にくいため、時計が遅くても減点しない。
**一杯主体の厩舎**は時計が出やすいため、時計だけで過信しない。

### 【厩舎別調教パターン一覧】

#### 馬なり主体（時計が出にくい）→ 時計遅くても減点禁止

| 厩舎 | 特徴 | 評価時の注意 |
|------|------|-------------|
| **千田輝彦** | ほとんど馬なりで無理をさせない | 時計が出なくても軽視禁止 |
| **藤沢和雄**（引退） | 馬なり主体、速い時計を出さない | 運動量は豊富、時計以外で判断 |
| **友道康夫** | 最終追い切りで芝・ポリトラックなど軽め | 休み明けでも走る、軽視禁止 |

#### 一杯主体（時計が出やすい）→ 時計だけで過信禁物

| 厩舎 | 特徴 | 評価時の注意 |
|------|------|-------------|
| **音無秀孝** | ほとんどの馬が一杯に追う | 時計は速いがラスト1Fに注目 |
| **西園正都** | 坂路で一杯に追う傾向 | 時計だけで判断しない |
| **森田直行** | ラストまで一杯に追われる | 消耗に注意 |

#### その他の特徴的な厩舎

| 厩舎 | 特徴 | 評価時の注意 |
|------|------|-------------|
| **堀宣行** | 土曜→水曜追い、日曜→木曜追い | 間隔が空いても買える |
| **国枝栄** | インコースを走る調教が多い | 時計以上には評価できない |
| **池江泰寿** | 朝イチの時間帯にびっしり | 好時計が出やすい時間帯 |
| **高野友和** | 馬場が荒れた時間帯を選ぶ | 時計が出にくい条件 |
| **清水久詞** | 調教本数が他厩舎より多め | 仕上がり重視 |

### 【厩舎別の追い切り評価補正】

| 厩舎タイプ | 追い切り時計が遅い場合 | 追い切り時計が速い場合 |
|------------|----------------------|----------------------|
| 馬なり主体 | **減点しない（±0）** | 好調サイン（+1点） |
| 一杯主体 | 要注意（-1点） | 過信禁物（±0） |
| 時計が出やすい条件 | 割引（-1点） | 額面通りに評価しない |

### 【厩舎別評価の確認手順】

1. **厩舎を確認**：馬なり主体か一杯主体かを確認
2. **追い切り内容を確認**：馬なり/強め/一杯どれか
3. **普段と比較**：その厩舎の馬としては動いているか
4. **ラスト1Fを確認**：終いの伸びが重要

### 【注意事項】

- 同じ厩舎でも馬によって調教パターンが異なる場合がある
- 追い切り時計だけでなく、**動きの質**（フットワーク、気配）も重要
- 陣営コメントと追い切り内容が一致しているか確認

## 追い切り評価の限界と注意事項（天皇賞秋2025反省より追加）

### 【基本原則】

追い切り評価が高くても、**展開や脚質が合わなければ凡走する**。
追い切り評価は「状態面」の指標であり、「適性」とは別問題。

### 【追い切り高評価でも警戒すべきケース】

| ケース | リスク | 対応 |
|--------|--------|------|
| 追い込み馬＋スローペース予想 | 届かない | 展開減点（-1〜2点） |
| 後方馬＋メンバーに差し馬多数 | 差し損ね | 展開減点（-1点） |
| 先行馬＋ハイペース予想 | 失速リスク | 展開減点（-1〜2点） |
| ラスト1Fが甘い傾向の馬 | 追い込み不発 | 脚質減点（-1点） |

### 【具体例：天皇賞秋2025】

| 馬名 | 追い切り | 人気 | 結果 | 敗因 |
|------|----------|------|------|------|
| メイショウタバル | **S** | 5番人気 | 6着 | 逃げて直線で甘くなる |
| ブレイディヴェーグ | **S〜A** | 4番人気 | 10着 | 後方待機、ラスト1F甘い |
| マスカレードボール | B | 1番人気 | **1着** | 中団から差し切り |

→ **追い切りS評価の2頭が凡走、B評価の馬が優勝**

### 【追い切り評価と展開の総合判断】

| 追い切り評価 | 展開向き | 最終評価 |
|-------------|---------|---------|
| S〜A | ◎ | 本命候補 |
| S〜A | △〜× | **過信禁物、評価据え置き** |
| B | ◎ | 有力候補 |
| B | △〜× | 押さえ程度 |
| C以下 | - | 軽視 |

### 【追い切り評価の限界チェックリスト】

分析完了後、以下を確認：

- [ ] 追い切りS〜A評価の馬の脚質と展開予測は合っているか？
- [ ] 追い込み馬がスローペース予想で過信していないか？
- [ ] ラスト1Fが甘い傾向の馬を追い切りだけで評価していないか？

## 長距離G1専用ルール（菊花賞2025反省より追加）

### 【対象レース】
- 菊花賞（京都3000m）
- 天皇賞・春（京都3200m）

| 条件 | 補正 |
|------|------|
| 2400m以下で折り合い不安の傾向 | **-3点**（危険な人気馬候補） |
| 距離延長幅500m以上＋折り合い不安 | **-4点** |

### 【折り合い不安のチェック項目】

- [ ] 過去レースで「行きたがった」「掛かった」のコメントあり

- [ ] テンション高めの馬（パドック情報）
- [ ] 初騎乗の騎手との組み合わせ

**ショウヘイ（菊花賞2025・14着/3番人気）**

- 距離実績：最長2400m（ダービー3着）

→ **追い切りS評価でも、折り合い不安で14着大敗**

## 馬場適性の明確評価（天皇賞春2025反省より追加）

### 【基本原則】

馬場適性は個体差が大きく、得意・不得意が明確な馬がいる。
特に「道悪巧者」が良馬場で走る場合、大幅に評価を下げる必要がある。

### 【馬場適性タイプの判定】

| タイプ | 判定条件 |
|--------|----------|
| **道悪巧者** | 稍重〜不良で勝利経験あり、良馬場で凡走傾向 |
| **良馬場巧者** | 良馬場で好走、道悪で成績低下傾向 |
| **万能型** | どちらでも安定した成績 |

### 【馬場不適合時の減点】

| 条件 | 補正 |
|------|------|
| 道悪巧者が良馬場（高速決着）で出走 | **-3点** |
| 良馬場巧者が稍重〜重で出走 | **-2点** |
| 良馬場巧者が不良で出走 | **-3点** |
| 「道悪の鬼」と呼ばれる馬が良馬場 | **危険な人気馬候補** |

### 【馬場適性の確認方法】

1. **過去成績の馬場別成績を確認**
   - 良馬場での成績（勝率・連対率）
   - 道悪での成績（勝率・連対率）

2. **騎手・陣営コメントを確認**
   - 「道悪は得意」「パンパンの良馬場がいい」等の発言

3. **血統から推測**
   - 父が道悪巧者（例：エピファネイア産駒は道悪◎）
   - 母系に重馬場実績

### 【具体例】

**ブローザホーン（天皇賞春2025・8着/5番人気）**
- タイプ：**道悪巧者**（「道悪の鬼」と呼ばれる）
- 当日馬場：良馬場・高速決着
- 騎手コメント：「柔らかい馬場でしぶとさ勝負の方がいい」
- 結果：8着敗退

→ **馬場適性不向きで「危険な人気馬」と見抜けた**

## 騎手のコミットメント評価（有馬記念2025反省より追加）

### 【基本原則】

騎手が複数の有力馬から選択できる状況で、特定の馬を選んだ場合、
その選択には**騎手の勝負勘・馬への信頼**が反映されている。
この「騎手のコミットメント」は評価において重要なシグナルとなる。

### 【コミットメントの種類】

| 種類 | 内容 | 評価 |
|------|------|------|
| **選択型** | 複数の有力馬から1頭を選んだ | 最も重要なシグナル |
| **継続型** | 長期間（1年以上）同じ馬に乗り続けている | 信頼関係の証 |
| **復帰型** | 他の騎手に乗り替わり後、再び戻ってきた | 馬への信頼 |
| **発言型** | 「この馬で勝ちたい」等の強いコメント | 騎手の本音 |

### 【コミットメント加点の条件と補正】

| 条件 | 補正 |
|------|------|
| 同レースに乗れる他の有力馬を**蹴って**選択 | **+3点** |
| 騎手が「悩んだ」と公言した上で選択 | **+2点**（本気の選択） |
| 同馬でG1勝利経験あり＋継続騎乗 | **+2点** |
| 「忘れられない」「最高」等の強い発言 | **+1点** |
| 2年以上の継続騎乗（主戦騎手） | **+1点** |

### 【具体例：有馬記念2025】

**戸崎圭太騎手のダノンデサイル選択**

| 項目 | 内容 |
|------|------|
| 選択肢1 | レガレイラ（昨年の有馬記念優勝、エリザベス女王杯優勝） |
| 選択肢2 | ダノンデサイル（ドバイシーマC優勝、ダービー馬） |
| **選択結果** | **ダノンデサイルを選択** |
| 騎手コメント | 「悩みました。本当ならどっちも乗りたい。体が2つ欲しい」 |
| 選択理由 | 「**ドバイでの走りが忘れられない。それくらいの手応えだった**」 |
| 馬への評価 | 「**かなりの能力を持ってると思っています**」 |

→ **適用加点：選択型（+3点）＋悩んだ上の選択（+2点）＋強い発言（+1点）= +6点相当**

### 【コミットメント評価の確認項目】

分析時に以下を確認：

- [ ] トップジョッキーが複数の有力馬に騎乗可能な状況か？
- [ ] その中で特定の馬を選んだ場合、選択理由を調べたか？
- [ ] 「悩んだ」等の発言があれば、本気の選択と判断したか？
- [ ] 騎手の馬に対するコメント（特に強い表現）を確認したか？
- [ ] 蹴られた側の馬の評価を相対的に下げたか？

### 【逆パターン：蹴られた馬の評価】

| 条件 | 補正 |
|------|------|
| 主戦騎手が同レースで他馬を選択 | **-1〜2点** |
| トップジョッキーが乗り替わりで降りた | **-1点** |
| 前走で好結果も騎手が乗り替わり | **要注意**（理由を調査） |

### 【注意事項】

- 騎手の選択は「スケジュール」の都合もあるため、理由を確認することが重要
- お手馬（長期継続騎乗）の場合は選択の余地がないことも
- 乗り替わりが決まった経緯（陣営の意向か騎手の意向か）を確認

## 追い切り評価の総合判定

### 【基本原則】

1. **複数媒体の評価を総合**して判定する（単一媒体に依存しない）
2. 数値基準は**参考情報**として活用
3. **脚質によって重視する指標が異なる**

### 【参照する媒体】

| 媒体 | 評価形式 | 特徴 |
|------|----------|------|
| **うまにてぃ** | S/A/B/C | 詳細なコメント付き |
| **うましる** | S/A/B/C | 全頭診断 |
| **東スポ** | A/B/C | トラックマン評価 |
| **競馬ラボ** | 独自評価 | 時計データ充実 |

### 【総合評価の算出方法】

| 各媒体評価 | 点数換算 |
|------------|----------|
| S | 5点 |
| A | 4点 |
| B | 3点 |
| C | 2点 |
| D | 1点 |

**総合評価 = 各媒体の平均点**

| 平均点 | 総合評価 | 補正 |
|--------|----------|------|
| 4.5以上 | **S** | +5点 |
| 3.5〜4.4 | **A** | +3点 |
| 2.5〜3.4 | **B** | ±0点 |
| 2.5未満 | **C** | -2点 |

### 【具体例：有馬記念2025 レガレイラ】

| 媒体 | 評価 | 点数 |
|------|------|------|
| うまにてぃ | B | 3点 |
| うましる | B | 3点 |
| 東スポ | B | 3点 |
| **平均** | | **3.0点 → B評価** |

→ 複数媒体が一致してB評価なら、信頼度が高い

## 数値基準（参考情報・脚質別）

### 【基本原則】

1. 媒体評価と**照合**するための参考情報

### 【脚質別の評価ポイント】

|------|----------|----------------|------|

## 金杯レース専用ルール（2021-2025年5年間検証より追加）

### 【検証結果サマリー】

**中山金杯**: 83点平均、的中率80%（優秀）
**京都金杯**: 59点平均、的中率20%（波乱多発）

5年間の検証から、金杯レースに特有のパターンが明らかになりました。
以下のルールを適用することで、予測精度の向上が期待できます。

---

### 【ルール1: 大外枠評価ルール】

#### 基本原則
18頭立ての大外枠（17-18枠）は、金杯のようなハンデ戦では**大幅な不利**となる。
特に中山金杯では直線が短く、外枠からの捲りが困難。

#### 減点ルール

| 条件 | 補正 |
|------|------|
| 18頭立て＋17枠または18枠 | **-4点** |
| 16頭立て＋16枠 | **-3点** |
| 15頭立て＋15枠 | **-2点** |

#### 具体例
**アルゴンウェル（2021年中山金杯）**
- 18枠（最大外）
- 3番人気だったが9着に敗退
- 大外枠で捲り届かず

→ **大外枠ルールで「危険な人気馬」と判定すべきだった**

---

### 【ルール2: 代替開催専用ルール】

#### 基本原則
京都金杯は京都競馬場改修工事により、中京競馬場での代替開催が多い（2021-2024年の4年間）。
代替開催時は、**中京コース適性を通常の1.5倍重視**する。

#### 評価方法

| 条件 | 対応 |
|------|------|
| 京都金杯が中京で代替開催 | 中京コース適性の点数を**1.5倍**で計算 |
| 中京で好走経験あり | **+2点** |
| 中京で勝利経験あり | **+3点** |
| 京都実績のみで中京未経験 | **-1点**（適性不明） |

#### 具体例
**コレペティトール（2024年京都金杯→中京開催）**
- 中京コース【1.0.0.2】と相性抜群
- 8番人気で優勝（3連単198,230円）
- 中京適性を重視すれば拾えた可能性

→ **代替開催ルールでコース適性を1.5倍評価すべきだった**

---

### 【ルール3: 高齢馬評価基準見直し】

#### 基本原則
金杯は「古馬ハンデ戦」であり、8歳以上の高齢馬も多数出走する。
従来は「高齢＝衰え」と評価しがちだったが、**G2以上の実績があれば能力維持**している可能性が高い。

#### 評価方法

| 条件 | 対応 |
|------|------|
| 8歳以上＋G2以上勝利経験あり | 実績評価を**4〜5点維持**（減点しない） |
| 8歳以上＋G3勝利経験あり | 実績評価を**3〜4点維持** |
| 8歳以上＋前走も好走（3着以内） | **状態良好と判断、+1点** |
| 9歳以上＋前走凡走 | **-2点**（流石に衰え） |

#### 具体例
**サクラトゥジュール（2025年京都金杯）**
- **8歳**
- G3勝利経験あり
- 6番人気で優勝
- 高齢だが能力維持していた

→ **高齢でも実績馬は軽視禁止**

---

### 【ルール4: 複合不利チェック強化】

#### 基本原則
**2つ以上の不利要素**を抱えている人気馬は、ハンデ戦では凡走リスクが高い。
不利要素を複数抱える人気馬は「危険な人気馬」候補とする。

#### 不利要素のチェックリスト

- [ ] 大外枠（17-18枠）
- [ ] 距離延長1000m以上
- [ ] 距離短縮1000m以上
- [ ] 休み明け（3ヶ月以上）
- [ ] コース替わり（東京→中山、京都→中京など）
- [ ] 前走凡走（5着以下）
- [ ] 斤量58kg以上（ハンデ戦で重い）
- [ ] 高齢（8歳以上）＋前走凡走

#### 判定ルール

| 不利要素の数 | 判定 |
|------------|------|
| **2つ以上** | **危険な人気馬候補** |
| **3つ以上** | **軽視推奨** |

#### 具体例
**某人気馬（仮）**
- 18枠（大外） ← 不利1
- 前走5着 ← 不利2
- 斤量58kg ← 不利3

→ **複合不利3つで「軽視」判定**

---

### 【ルール5: 距離大幅変更減点強化】

#### 基本原則
前走から**1000m以上の距離変更**がある馬は、適性が未知数。
特に短縮1000m以上はスピード不足、延長1000m以上はスタミナ不足のリスクが高い。

#### 減点ルール

| 条件 | 補正 |
|------|------|
| 距離延長**1000m以上** | **-3点** |
| 距離延長**800m以上** | **-2点** |
| 距離短縮**1000m以上** | **-3点** |
| 距離短縮**800m以上** | **-2点** |
| 距離延長＋過去に同距離好走経験あり | **減点なし**（適性確認済み） |

#### 具体例
**前走1600m→今回2600m（+1000m）**
- 距離延長1000m → **-3点**
- ただし、過去に2400m以上で好走経験があれば減点なし

---

### 【ルール6: ハンデ戦波乱補正】

#### 基本原則
金杯のようなハンデ戦では、**人気薄の軽ハンデ馬**が激走することが多い。
特に以下の条件を満たす馬は穴馬候補として浮上する。

#### 加点ルール

| 条件 | 補正 |
|------|------|
| **10番人気以下＋56kg以下＋前走5着以内** | **+2点**（穴馬候補） |
| **10番人気以下＋54kg以下＋前走3着以内** | **+3点**（有力穴馬） |
| 人気薄＋軽ハンデ＋追い切りA以上 | **穴馬候補に浮上** |

#### 具体例
**リカンカブール（2024年中山金杯）**
- 5番人気、斤量56kg
- 前走2着（好走）

### 【ルール7: 前走凡走からの巻き返し評価】

#### 基本原則
前走で着順が悪くても、**上がり3Fが上位**（5位以内）であれば、次走での巻き返しが期待できる。
着順だけで判断せず、**レース内容**を精査することが重要。

#### 加点ルール

| 条件 | 補正 |
|------|------|
| 前走6着以下＋上がり3F **1〜2位** | **+3点**（巻き返し候補） |
| 前走6着以下＋上がり3F **3〜5位** | **+2点** |
| 上記＋今回追い切りA以上 | **穴馬候補に浮上** |
| 前走で不利あり（出遅れ、包まれ等）＋上がり上位 | **+1点**（追加加点） |

#### 具体例
**某馬（仮）**
- 前走8着（凡走）
- 上がり3F: メンバー中2位（33.5秒）
- 今回追い切りA評価

→ **前走不利＋上がり上位＋追い切り良好で巻き返し候補**
