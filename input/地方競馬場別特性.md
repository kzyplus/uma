## 大井競馬場

### 基本情報
| 項目         | 内容                    |
| ------------ | ----------------------- |
| 所在地       | 東京都品川区            |
| コース       | 右回り                  |
| 外回りコース | 1周1,600m               |
| 内回りコース | 1周1,400m               |
| 直線距離     | 386m（外）/ 286m（内）  |
| 砂厚         | 約9cm                   |

### コース特徴
- **南関東最大の競馬場**
- 外回り直線386mは**南関東最長**
- 内回りは小回りで先行有利
- **スピード持続力**が問われる
- ナイター開催が中心（トゥインクルレース）

### 距離別特徴

#### ダート1000m（内回り）
- スタート後すぐにコーナー
- **逃げ・先行馬有利**
- テンのスピードが重要

#### ダート1200m（外回り）
- 向正面スタート
- 先行有利だが、直線で差しも届く
- スピード勝負

#### ダート1400m（内回り）
- 内回りコースを使用
- **先行馬有利**
- 器用さが求められる

#### ダート1600m（外回り）★主要距離
- スタンド前からスタート
- 直線386mで**差し・追込も届く**
- ペース次第で展開が変わる
- 東京シンデレラマイル、マイルグランプリの舞台

#### ダート1800m（外回り）
- バランスの良いコース
- スタミナも必要
- 先行〜差しまで幅広く

#### ダート2000m（外回り）
- 東京大賞典、帝王賞の舞台
- **総合力**が問われる
- スタミナ＋スピード持続力

### 枠順傾向
| 距離 | 傾向 |
|------|------|
| 1000m | 内枠有利 |
| 1200m | やや内枠有利 |
| 1400m | 内枠有利 |
| 1600m | **内枠やや有利、外枠も好走あり** |
| 1800m | 枠順影響小さい |
| 2000m | 枠順影響小さい |

### 馬場傾向
| 馬場状態 | 傾向                   |
| -------- | ---------------------- |
| 良       | 時計速め、スピード勝負 |
| 稍重     | やや時計かかる         |
| 重・不良 | パワー型有利、道悪巧者浮上 |

## 川崎競馬場

### 基本情報
| 項目         | 内容               |
| ------------ | ------------------ |
| 所在地       | 神奈川県川崎市     |
| コース       | 左回り             |
| コース       | 1周1,200m          |
| 直線距離     | 300m               |
| 砂厚         | 約11cm             |

### コース特徴
- **南関東唯一の左回り**
- 1周1,200mの**小回りコース**
- 直線300mはやや短め
- **先行馬有利**の傾向が強い
- 砂が深めでパワー型向き

### 距離別特徴

#### ダート900m
- 最短距離、スピード勝負
- **逃げ馬有利**
- テンのダッシュ力が重要

#### ダート1400m★主要距離
- 川崎の主要距離
- **先行馬有利**
- 小回りで器用さも必要
- 川崎記念の舞台

#### ダート1500m
- やや中途半端な距離
- 先行有利

#### ダート1600m
- エンプレス杯の舞台
- スタミナも必要
- 先行〜好位差し

#### ダート2000m
- スタミナ必須
- 折り合い重要
- 先行〜差し

#### ダート2100m
- 川崎記念の舞台
- 長距離適性必要
- 総合力勝負

### 枠順傾向
| 距離 | 傾向 |
|------|------|
| 900m | 内枠有利 |
| 1400m | **内枠有利** |
| 1500m | 内枠やや有利 |
| 1600m | 内枠やや有利 |
| 2000m以上 | 枠順影響小さい |

### 馬場傾向
| 馬場状態 | 傾向                     |
| -------- | ------------------------ |
| 良       | 時計標準                 |
| 稍重以下 | **パワー型有利、前残り** |

### 左回り適性
- JRA東京・中京・新潟出身馬に注意
- **右回り慣れした馬は要注意**
- 左回り実績を確認すること

## 浦和競馬場

### 基本情報
| 項目         | 内容               |
| ------------ | ------------------ |
| 所在地       | 埼玉県さいたま市   |
| コース       | 右回り             |
| コース       | 1周1,200m          |
| 直線距離     | 220m               |
| 砂厚         | 約9cm              |

### コース特徴
- **南関東最小の競馬場**
- 1周1,200mの超小回り
- 直線220mは**非常に短い**
- **逃げ・先行馬が断然有利**
- 差し・追込は極めて届きにくい

### 距離別特徴

#### ダート800m
- 最短距離
- **完全に逃げ馬有利**
- 一瞬のスピード勝負

#### ダート1400m★主要距離
- 浦和の主要距離
- **逃げ・先行馬有利**
- 差しはほぼ届かない
- 浦和記念の舞台

#### ダート1500m
- 先行有利
- 直線短く追込不発

#### ダート1600m
- やや距離延長
- 先行有利

#### ダート2000m
- 浦和記念の舞台
- **スタミナ必須だが先行有利**
- 小回り適性も重要

### 枠順傾向
| 距離 | 傾向 |
|------|------|
| 800m | **内枠圧倒的有利** |
| 1400m | **内枠有利** |
| 1500m | 内枠有利 |
| 1600m以上 | 内枠やや有利 |

### 馬場傾向
| 馬場状態 | 傾向                       |
| -------- | -------------------------- |
| 良       | 時計速め                   |
| 稍重以下 | パワー型有利、さらに前残り |

### 浦和の特殊性
- **差し・追込馬は大幅減点**すべき
- 前走で差して好走した馬も、浦和では不発の可能性
- **先行力のある馬を優先**

### 逃げ・先行馬が強い場合
- **浦和 ＞ 船橋 ＞ 川崎 ＞ 大井**

### 差し・追込馬が届く場合
- **大井 ＞ 川崎 ＞ 船橋 ＞ 浦和**

- **川崎のみ**（他3場は右回り）
