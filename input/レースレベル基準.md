# レースレベル基準データベース

## 概要
レースのレベル（相手関係の強さ）を評価するための基準です。
着順だけでなく、どのレベルのレースで何秒差だったかを重視します。

## レースレベル評価ランク

### 重賞レベル評価

#### S（最高レベル）
- G1優勝馬が複数出走
- その年の代表馬が出走
- 例：宝塚記念、有馬記念

#### A（高レベル）
- G1馬が出走
- G2上位クラス
- 例：大阪杯、毎日王冠

#### B+（中上位レベル）
- G2標準、G3上位
- 重賞勝ち馬複数
- 例：金鯱賞、中山記念

#### B（中レベル）
- G3標準
- OP勝ち馬が複数
- 例：函館記念、小倉記念

#### B-（中下位レベル）
- G3下位、L上位
- 例：ローカル重賞

#### C（標準レベル）
- OP/L標準
- 例：一般OP特別

#### D（低レベル）
- 条件戦レベル
- メンバーが手薄

## 着差評価基準

### 着差の評価（勝ち馬との差）
| 着差                   | 評価 | コメント     |
| ---------------------- | ---- | ------------ |
| 0.0秒（同着～ハナ）    | ◎    | 勝ちに等しい |
| 0.1秒（クビ）          | ◎    | ほぼ互角     |
| 0.2秒（クビ～半馬身）  | ○    | 好勝負       |
| 0.3秒（半馬身～1馬身） | ○    | 善戦         |
| 0.4～0.5秒（1～2馬身） | ○    | 力差小さい   |
| 0.6～0.8秒（2～4馬身） | △    | やや力差あり |
| 0.9～1.2秒（5～7馬身） | △    | 力差あり     |
| 1.3秒以上（大差）      | ×    | 力差大きい   |

## 使用方法

1. **過去レースの評価時**
   - レースのクラス（G1/G2/G3/OP等）を確認
   - 着差（秒差）を確認
   - 上記表に照らして評価

2. **例：函館記念出走馬の評価**
   - 前走G2で0.3秒差5着 → A評価相当（G2で善戦）
   - 前走G3で1.0秒差8着 → C評価相当（やや力差あり）
   - 前走OPで0.1秒差2着 → B評価相当（OP上位）
