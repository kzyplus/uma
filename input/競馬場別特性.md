## 函館競馬場

### 基本情報
| 項目         | 内容                       |
| ------------ | -------------------------- |
| 所在地       | 北海道函館市               |
| コース       | 右回り                     |
| 芝コース     | 1周1,626.6m                |
| ダートコース | 1周1,475.8m                |
| 直線距離     | 262m（芝）/ 260m（ダート） |
| 高低差       | 3.5m                       |
| 芝種類       | 洋芝（オーバーシード）     |

### コース特徴
- **小回り**で器用さが求められる
- **直線が短い**ため、先行馬有利の傾向
- **洋芝**のため時計がかかりやすい
- **スピード持続力**が重要
- 3～4コーナーがスパイラルカーブ

### 距離別特徴

#### 芝1200m
- スタート後すぐにコーナー
- 内枠有利、逃げ・先行馬有利
- 器用な立ち回りが必要

#### 芝1800m
- スタンド前からスタート
- 先行馬有利だが、差しも届く
- ペース次第で展開が変わる

#### 芝2000m
- 向正面からスタート
- コーナー4回、スタミナも必要
- 先行力＋持続力がカギ

#### 芝2600m
- 長距離戦、スタミナ重視
- ペースが落ち着きやすい
- 差し馬にもチャンス

### 有利な血統傾向
- **父**: ディープインパクト、ロードカナロア、キタサンブラック
- **母父**: サンデーサイレンス系、ノーザンダンサー系
- 洋芝適性のある欧州型血統も好走

### 馬場傾向
| 馬場状態 | 傾向                 |
| -------- | -------------------- |
| 良       | 時計は標準～やや遅め |
| 稍重     | パワー型有利に       |
| 重・不良 | 道悪巧者の台頭       |

### 枠順傾向（芝）
- 1200m: 内枠有利
- 1800m: 枠順の有利不利は小さい
- 2000m: やや内枠有利
- 2600m: 枠順の影響は小さい

#### 芝2600m

## 新潟競馬場

### 基本情報
| 項目           | 内容                               |
| -------------- | ---------------------------------- |
| 所在地         | 新潟県新潟市                       |
| コース         | 左回り（外回り）/ 右回り（内回り） |
| 芝コース（外） | 1周2,223m                          |
| 芝コース（内） | 1周1,623m                          |
| 直線距離（外） | 658.7m                             |
| 直線距離（内） | 358.7m                             |
| 高低差         | ほぼ平坦                           |

### コース特徴
- **外回りは日本最長の直線**（658.7m）
- **完全平坦**でスピード決着
- 外回りは**差し・追込有利**
- 内回りは先行有利
- 直線1000mコースあり（唯一）

### 有利な血統傾向
- **外回り**: 末脚のキレる血統（ディープインパクト系）
- **内回り**: スピード持続型
- **直線**: 純粋なスピード血統

- **長い直線**と**緩やかな坂**
- **末脚の切れ味**が問われる
- 差し・追込が届きやすい
- **瞬発力勝負**になりやすい

- 向正面スタート
- スピードと持続力のバランス

- 緩い下り坂からスタート

- スタート直後にコーナー

#### 芝2000m
- 秋天皇賞の舞台
- バランスの良い適性が必要

#### 芝2400m
- 日本ダービー、JCの舞台
- スタミナ＋瞬発力

#### 芝2500m
- 有馬記念の代替舞台にもなる
- 長距離適性必要

- **父**: ディープインパクト、キングカメハメハ、ハーツクライ
- **母父**: サンデーサイレンス系
- 瞬発力に優れた血統が有利

### 馬場傾向
| 馬場状態 | 傾向                 |
| -------- | -------------------- |
| 良       | 高速馬場になりやすい |

| 直線距離     | 403.7m（外）/ 328.4m（内）      |

### コース特徴
- **3コーナーに坂**（淀の坂）

- 直線は**平坦**

#### 芝1200m（内回り）

#### 芝1400m（外回り）
- 坂の下りからの仕掛け

#### 芝1600m（外回り）
- マイルCS、桜花賞の舞台
- 坂を下ってからの直線勝負

#### 芝1800m
- 3歳戦で多用

#### 芝2000m（内回り）
- 秋華賞の舞台
- 器用さ重視

#### 芝2200m（外回り）
- エリザベス女王杯の舞台
- 坂の使い方がポイント

#### 芝2400m
- 菊花賞と同距離
- スタミナ必要

#### 芝3000m以上
- 天皇賞（春）、菊花賞の舞台
- 長距離適性必須

### 有利な血統傾向
- **父**: ディープインパクト、ステイゴールド系
