# 地方競馬分析指示書

## 分析の基本方針

### 最重要事項
- **正確な情報を最優先**する
- 不明な情報は「不明」と明記し、推測は「※推測」と明示する
- 根拠のない断定は避ける

### JRAとの主な違い
| 項目 | JRA | 地方競馬 |
|------|-----|----------|
| 追い切り情報 | 栗東・美浦で充実 | **陣営コメント重視** |
| コース | 芝・ダート両方 | **ダート中心** |
| 騎手データ | 統計豊富 | 所属別に確認 |
| レースレベル | G1〜未勝利の階層 | 地区・グレード別 |

## 評価基準（12項目・基本60点）

### 1. 着差重視の過去実績（/5点）
| 点数 | 基準                                       |
| ---- | ------------------------------------------ |
| 5点  | ダートグレード（JpnI〜III）で0.3秒差以内   |
| 4点  | 南関東重賞（SI〜SIII）で0.5秒差以内        |
| 3点  | 準重賞・オープンで好走                     |
| 2点  | 条件戦で善戦レベル                         |
| 1点  | 目立った実績なし                           |

### 2. レースレベル（相手関係）（/5点）
| 点数 | 基準                                 |
| ---- | ------------------------------------ |
| 5点  | JpnI・JRA交流重賞で好走経験あり      |
| 4点  | JpnII〜III・南関東SIで好走           |
| 3点  | 南関東SII〜SIII・準重賞レベルで好走  |
| 2点  | 条件戦レベルが主戦場                 |
| 1点  | 相手関係で見劣り                     |

### 3. 血統（産駒傾向）（/5点）
| 点数 | 基準                                    |
| ---- | --------------------------------------- |
| 5点  | 父・母父ともにダート・距離適性◎         |
| 4点  | 父または母父がダート・距離適性◎         |
| 3点  | 血統的に問題なし                        |
| 2点  | 芝血統でダート適性に疑問                |
| 1点  | 血統的に不向き                          |

#### 地方ダート向き種牡馬
- **ヘニーヒューズ** - ダート短距離◎、1600m◎
- **シニスターミニスター** - ダート◎、道悪◎
- **ダノンレジェンド** - ダート中距離◎
- **ゴールドアリュール系** - ダート全般◎
- **マジェスティックウォリアー** - ダート◎

| ---- | -------------------------------------- |

| 4点  | 同コースで好走、または類似コースで実績 |
| 3点  | 初コースだが適性は問題なさそう         |
| 2点  | コース適性にやや不安                   |
| 1点  | コース適性に大きな不安                 |

### 5. 輸送適性（/5点）
| 点数 | 基準                               |
| ---- | ---------------------------------- |
| 5点  | 南関東内移動、または輸送実績豊富   |
| 4点  | 他地区からの遠征で好走経験あり     |
| 3点  | 輸送経験少ないが、問題なさそう     |
| 2点  | 遠征で成績落ちる傾向あり           |
| 1点  | 遠征で明らかにパフォーマンス低下   |

※南関東4場（大井・川崎・船橋・浦和）間は輸送負担軽微

## 穴馬スクリーニング

### 評価順位 vs 人気順位のギャップ
| 条件 | 判定 |
|------|------|
| 評価順位が人気順位より**3つ以上高い** | **穴馬候補** |
| 評価順位が人気順位より**5つ以上高い** | **有力穴馬候補** |

### 地方競馬特有の穴馬パターン

## 地方重賞（SI〜SIII）固有の注意点

### 騎手の勢い ★重要★
- **直近1週間の重賞成績を必ず確認**
- 連勝中・好調の騎手は **+2点** 加点
- 特に年末年始の連続開催では「波に乗った騎手」が連勝しやすい
- 例：矢野貴之騎手の東京大賞典→東京シンデレラマイル連勝（2025年）

### 若い馬の勢い（3歳馬 vs 古馬）
- 地方SIIIでは3歳馬が古馬を圧倒するケースあり
- 3歳重賞勝ち馬は **+2点** 加点を検討
- 特に下半期（10月以降）の3歳馬は成長著しい
- 斤量差（3歳54kg vs 古馬57kg = 3kg差）も有利に働く

### 斤量差の影響
- JpnIと異なり、**SIIIでは斤量差の影響が大きい**
- 3kg以上の差がある場合は軽量馬に **+1〜2点** 調整
- 別定戦では実績馬の斤量増に注意

### 馬場状態の当日確認 ★必須★
- 地方競馬は馬場悪化で傾向が激変する
- **良馬場 → 先行有利**
- **稍重以下 → 差し馬台頭の可能性**
- 当日の馬場発表を必ず確認してから最終判断

### 荒れやすいレースの認識
- 東京シンデレラマイル、東京2歳優駿牝馬など**牝馬限定戦は荒れやすい**
- 16番人気2着など大穴も発生する
- 穴馬の網を広げ、3連単は手広く
