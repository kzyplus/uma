## 💡 改善提案: 4つの重要ルール追加

### 提案1: 大外枠評価ルール ★最優先★

**背景:**
2025年中山金杯で、18頭立て18番（大外枠）のホウオウビスケッツ（2番人気）が16着大敗。

**新規ルール:**
```
【18頭立て大外枠評価】
- 17番・18番枠: 枠順評価を1点に減点（-4点）
- さらに重斤量58.0kg以上の場合: 追加で-1点
- 複合不利（大外枠 + 重斤量 + 18頭立て）の場合: "危険な人気馬"候補に自動追加
```

**適用例:**
- ホウオウビスケッツ: 18番枠（-4点）+ 59.5kg（-1点） = 合計-5点 → 危険視すべきだった

---

### 提案2: 代替開催専用ルール

**背景:**
京都→中京の代替開催では、通常の京都金杯とは異なる適性が求められる。

**新規ルール:**
```
【代替開催時のコース適性評価】
- 代替開催の場合、「代替先コース適性」を1.5倍評価
- 京都→中京の場合: 中京芝1600m実績を重視（京都芝1600m実績の1.2倍評価）
- 左回り適性馬を+1点加点（中京は左回り、京都は右回り）
```

**適用例:**
- 2023年イルーシヴパンサー: 左回り適性+1点 → A評価で拾えた
- 2025年サクラトゥジュール: 中京1600m適性を重視していれば評価アップ

---

### 提案3: 高齢馬評価基準の見直し

**背景:**
2025年京都金杯で8歳のサクラトゥジュールが優勝。「8歳以上=実績3点」の固定評価が裏目に出た。

**新規ルール:**
```
【8歳以上の馬の評価】
- 基本: 実績評価3点（従来通り）
- ただし以下の条件を満たす場合は実績評価4-5点を許容:
  1. 過去にG2以上で3着以内の実績がある
  2. 前走で1着または0.3秒差以内の2着
  3. 去勢明け・調教◎の場合
  4. 近3走で連対率50%以上
```

**適用例:**
- サクラトゥジュール（8歳）: G3勝ち馬 + 調教○ → 実績4点に引き上げ可能

---

### 提案4: 複合不利チェックの強化

**背景:**
単一の不利要因だけでなく、複数の不利が重なる場合の減点が不十分。

**新規ルール:**
```
【複合不利チェック】
以下の不利要因が2つ以上重なる場合、「危険な人気馬」候補に自動追加:
1. 大外枠（17-18番）
2. 重斤量（58.0kg以上）
3. 18頭立て
4. 展開不向き（逃げ馬でハイペース予想など）
5. 追い切り評価B以下
6. 休み明け4ヶ月以上
```

**適用例:**
- 2025年ホウオウビスケッツ: ①大外18番 + ②59.5kg + ③18頭立て → 3つ該当で危険視

## 🎯 最終結論

### ✅ 3年間で成功した点

1. **中山金杯での高精度**:
   - 3年間で2回本命的中（2023, 2024）
   - 2025年も評価2位が1着で「惜しい的中」
   - 中山コースの特性（内有利・坂）を正確に評価

2. **2024年中山で3連複16,510円的中**:
   - 12項目評価基準の有効性を実証
   - 追い切り評価 + 内有利馬場予測 + 危険な人気馬検出が完璧に機能

3. **騎手・血統・展開評価の安定性**:
   - 戸崎圭太（中山◎）、M.デムーロ（実力◎）等トップ騎手の評価は高精度
   - キングカメハメハ系、ハーツクライ産駒の適性評価は安定

4. **危険な人気馬検出**:
   - 2024年中山でエピファニー（1番人気）の11着大敗を的中
   - 折り合い不安・展開不向きの減点ルールが機能

### ❌ 3年間で改善が必要な点

1. **京都金杯での精度低下**:
   - 3年間で本命的中0回
   - 穴馬（5～8番人気）の検出失敗が目立つ
   - 代替開催（京都→中京）時の評価が不十分

2. **2025年の精度低下**:
   - 平均スコア62.5点（前年比-12.5点）
   - 本命的中率0%、回収率0%
   - 複合不利の見逃し（大外枠+重斤量）が致命的

3. **穴馬スクリーニングの不足**:
   - 3年平均で穴馬検出率33%
   - 5番人気以下の馬の評価精度が低い
   - 「評価順位 vs 人気順位のギャップ」検出が不徹底

4. **実績評価の硬直性**:
   - 高齢馬（8歳以上）を一律減点
   - 重賞未勝利馬を過小評価
   - 去勢明け・前走不利等の巻き返し評価が不足

### 📌 総合評価: **B（70.8点）**

現行12項目評価基準は、**中山金杯では高い精度（80点）を示し、基本的な評価フレームワークは有効**であることが3年間の検証で実証されました。

ただし、**京都金杯（特に代替開催時）では精度が低く（65点）**、以下の4つの改善が急務です:

1. **大外枠評価ルール**（18頭立て17-18番: -4点、重斤量併用で-1点追加）
2. **代替開催専用ルール**（代替先コース適性1.5倍、左回り+1点）
3. **高齢馬評価基準見直し**（8歳以上でもG2以上実績あれば実績4-5点許容）
4. **複合不利チェック強化**（2つ以上の不利要因で危険な人気馬候補に自動追加）

これらの改善を進めることで、**目標A評価80点以上の達成**が期待できます。
