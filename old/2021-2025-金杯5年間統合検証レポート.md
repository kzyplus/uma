## 💡 改善提案: 7つの重要ルール追加

### 提案1: 大外枠評価ルール ★最優先★

**背景:**
2025年中山金杯で、18頭立て18番（大外枠）のホウオウビスケッツ（2番人気）が16着大敗。

**新規ルール:**
```
【18頭立て大外枠評価】
- 17番・18番枠: 枠順評価を1点に減点（-4点）
- さらに重斤量58.0kg以上の場合: 追加で-1点
- 複合不利（大外枠 + 重斤量 + 18頭立て）の場合: "危険な人気馬"候補に自動追加
```

---

### 提案2: 代替開催専用ルール

**背景:**
京都→中京の代替開催は5年中4回発生し、通常の京都金杯とは異なる適性が求められる。

**新規ルール:**
```
【代替開催時のコース適性評価】
- 代替開催の場合、「代替先コース適性」を1.5倍評価
- 京都→中京の場合: 中京芝1600m実績を重視（京都芝1600m実績の1.2倍評価）
- 左回り適性馬を+1点加点（中京は左回り、京都は右回り）
```

---

### 提案3: 高齢馬評価基準の見直し

**背景:**
2025年京都金杯で8歳のサクラトゥジュールが優勝。「8歳以上=実績3点」の固定評価が裏目に出た。

**新規ルール:**
```
【8歳以上の馬の評価】
- 基本: 実績評価3点（従来通り）
- ただし以下の条件を満たす場合は実績評価4-5点を許容:
  1. 過去にG2以上で3着以内の実績がある
  2. 前走で1着または0.3秒差以内の2着
  3. 去勢明け・調教◎の場合
  4. 近3走で連対率50%以上
```

---

### 提案4: 複合不利チェックの強化

**背景:**
単一の不利要因だけでなく、複数の不利が重なる場合の減点が不十分。

**新規ルール:**
```
【複合不利チェック】
以下の不利要因が2つ以上重なる場合、「危険な人気馬」候補に自動追加:
1. 大外枠（17-18番）
2. 重斤量（58.0kg以上）
3. 18頭立て
4. 展開不向き（逃げ馬でハイペース予想など）
5. 追い切り評価B以下
6. 休み明け4ヶ月以上
```

---

### 提案5: 前走凡走からの巻き返し評価 ★重要★

**背景:**
2021年京都金杯でケイデンスコール（前走6着）が12番人気で優勝。前走凡走馬の復調を評価できず。

**新規ルール:**
```
【前走不利→巻き返し評価】
前走6着以下でも、以下の条件を満たす場合は+2点加点:
1. 前走で上がり3F上位（5位以内）
2. 前走で明確な不利（出遅れ、前が詰まる、包まれる等）
3. 前走で斤量過多（今回-3kg以上の減量）
4. 調教◎かつ陣営コメントが前向き
```

**適用例:**
- 2021年ケイデンスコール: 前走6着でも上がり3F上位 → +2点加点で評価アップ

---

### 提案6: 距離大幅変更の減点強化

**背景:**
2022年中山金杯でトーセンスーリヤ（距離短縮1600m: 3600m→2000m）が5着敗退。

**新規ルール:**
```
【距離変更評価の厳格化】
- 距離短縮1000m以上: -2点
- 距離短縮1500m以上: -3点
- 距離延長500m以上: -1点
- 距離延長800m以上: -2点
```

---

### 提案7: ハンデ戦の波乱補正

**背景:**
2021年京都金杯で人気薄馬（12番人気・14番人気）が馬券圏内。ハンデ戦の斤量差メリットを過小評価。

**新規ルール:**
```
【ハンデ戦の人気薄補正】
10番人気以下の馬で、以下の条件を満たす場合は+2点加点:
1. 斤量54.0kg以下
2. 前走で5着以内
3. コース適性○以上
4. 血統適性○以上
```

**適用例:**
- 2021年ケイデンスコール（12番人気・斤量54kg）: +2点加点で評価アップ

## 🎯 最終結論

### ✅ 5年間で成功した点

1. **中山金杯での高精度を維持**:
   - 5年間で4回本命的中（2021, 2022, 2023, 2024）
   - 本命的中率80%は実用レベル
   - 2025年も評価2位が1着で「惜しい的中」
   - 中山コースの特性（内有利・坂）を正確に評価

2. **高配当馬券の的中実績**:
   - 2021年中山: 3連単64,390円的中
   - 2024年中山: 3連複16,510円的中（3連単84,690円）
   - 12項目評価基準で高配当獲得可能と実証

3. **騎手・血統・展開評価の安定性**:
   - 戸崎圭太（中山◎）、松山弘平（実力◎）、M.デムーロ（実力◎）の評価は高精度
   - キングカメハメハ系、ハーツクライ産駒の適性評価は安定
   - 内有利・ハイペース予測の的中率高い

4. **危険な人気馬検出の高精度**:
   - 2021年中山: ディープボンド（3番人気）14着的中
   - 2022年中山: シャムロックヒル（3番人気）7着的中
   - 2024年中山: エピファニー（1番人気）11着的中
   - 危険な人気馬的中率60%（実施4回中3回的中）

### ❌ 5年間で改善が必要な点

1. **京都金杯での精度低下**:
   - 5年間で本命的中0回（的中率20%）
   - 5年平均スコア59点（中山83点 vs 京都59点）
   - 穴馬（5～12番人気）の検出失敗が目立つ
   - 代替開催（京都→中京）が5年中4回で評価が不十分

2. **2025年の精度低下**:
   - 平均スコア62.5点（過去5年最低）
   - 本命的中率0%、回収率0%
   - 複合不利の見逃し（大外枠+重斤量）が致命的

3. **穴馬スクリーニングの不足**:
   - 5年平均で穴馬検出率30%（目標50%）
   - 特に京都金杯で課題（5年平均スコア59点）
   - 5番人気以下の馬の評価精度が低い

4. **前走凡走馬の巻き返し評価**:
   - 2021年京都: ケイデンスコール（前走6着→12番人気1着）を評価7位で大外し
   - 上がり3F上位等のポジティブ要因を見逃す

5. **実績評価の硬直性**:
   - 高齢馬（8歳以上）を一律減点
   - 重賞未勝利馬を過小評価
   - 去勢明け・前走不利等の巻き返し評価が不足

### 📌 総合評価: **B-（69.5点）**

現行12項目評価基準は、**中山金杯では極めて高い精度（5年平均83点・本命的中率80%）**を示し、基本的な評価フレームワークは有効であることが5年間の検証で実証されました。

ただし、**京都金杯（特に代替開催時）では精度が低く（5年平均59点・本命的中率20%）**、以下の7つの改善が急務です:

1. **大外枠評価ルール**（18頭立て17-18番: -4点、重斤量併用で-1点追加）
2. **代替開催専用ルール**（代替先コース適性1.5倍、左回り+1点）
3. **高齢馬評価基準見直し**（8歳以上でもG2以上実績あれば実績4-5点許容）
4. **複合不利チェック強化**（2つ以上の不利要因で危険な人気馬候補に自動追加）
5. **前走凡走からの巻き返し評価**（上がり3F上位なら+2点）
6. **距離大幅変更の減点強化**（1000m以上変更は-2～-3点）
7. **ハンデ戦の波乱補正**（10番人気以下+軽量なら+2点）

これらの改善を進めることで、**目標A評価80点以上の達成**が期待できます。
