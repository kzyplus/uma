# 🏇 競馬予想シート生成システム

## 📋 プロジェクト概要

競馬レースの予想データを視覚的にわかりやすく表示するウェブアプリケーションのテンプレートシステムです。

**現在のバージョン**: ホープフルステークス2025（GⅠ）

## ✨ 主な機能

### 1. 視覚的なデータ表示
- **総合評価グラフ**: Chart.jsを使用した上位10頭の評価点バーチャート
- **レーダーチャート**: 各馬の12項目評価を視覚化
- **レスポンシブデザイン**: PC・スマートフォン両対応
- **落ち着いたカラースキーム**: 長時間閲覧しても疲れにくい配色

### 2. 情報の整理
- **基本情報**: 開催日・競馬場・コース・馬場状態・天候を一目で確認
- **コース特性**: コース特徴を詳細に説明
- **最終結論セクション**: 本命・対抗・穴馬・危険な人気馬を明確に表示
- **総合評価ランキング**: 全馬を評価点順にソート
- **個別馬評価**: 詳細な評価項目と良い点・不安材料・期待点を整理
- **展開予測**: ペース・脚質・隊列イメージを表示
- **買い目参考**: 推奨馬券を提示

### 3. ユーザビリティ
- **フィルター機能**: 「全て表示」「上位馬のみ」「穴馬候補」でフィルタリング可能
- **カラーコーディング**: 馬場状態の適性を色分け（◎赤/○緑/△黄）
- **ホバーエフェクト**: カードにマウスオーバーで視覚的フィードバック
- **高さ制御**: グラフの高さを固定してスクロール問題を解決

## 🗂️ ファイル構成

```
├── index.html              # メインHTMLファイル（テンプレート）
├── data-template.js        # データ入力用テンプレート
├── TEMPLATE_GUIDE.md       # 詳細な使い方ガイド
└── README.md              # このファイル
```

## 🚀 クイックスタート

### 新しいレースの予想シートを作成する

1. **`index.html`をコピー**して新しいファイルを作成
2. **`data-template.js`を開いて**馬データを入力
3. **`TEMPLATE_GUIDE.md`を参照**して各セクションを編集
4. ブラウザでプレビュー確認
5. Publishタブからデプロイ

### 最小限の編集箇所

1. ✅ レース名（タイトル）
2. ✅ 基本情報（開催日、競馬場、コース、馬場状態、天候）
3. ✅ 最終結論（本命・対抗・穴馬・危険馬）
4. ✅ JavaScriptデータ（`horses`配列と`rankingData`配列）
5. ✅ 買い目参考

## 📊 評価システム

### 12項目評価（各5点満点、合計60点）
1. 着差重視の過去実績
2. レースレベル（相手関係）
3. 血統（産駒傾向）
4. コース適性
5. 輸送適性
6. 休み明け実績
7. 脚質と展開相性
8. 枠順と脚質相性
9. 騎手との相性
10. 調教・仕上がり
11. 馬体の状態
12. 厩舎との相性

### バッジシステム
- **本命** (`honmei`) - ピンク色
- **対抗** (`taikou`) - ブルー色
- **穴馬** (`ana`) - グリーン色
- **危険な人気馬** (`kiken`) - オレンジ色

### 適性表示
- **◎** - 非常に良い
- **○** - 良い
- **△** - 微妙
- **×** - 悪い
