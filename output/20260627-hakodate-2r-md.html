予想信頼度

TIER2（中信頼）（最高47pt / 1-2位点差3pt / 12頭）
- 平場かつ全頭未勝利のため上限はTIER2。
- 上位2頭（◎○）は人気・能力とも抜けるが、◎は0勝（3着止まりが多い）・○は1000m未経験という不確定要素あり。

コース特性（函館ダート1000m）

  • 右・小回り・スタートから3角まで短い前残り[強]コース（実測: 前位置取り60.2% / 後方4.1%）。
  • 早い流れで先行・好位を取れた馬がそのまま残る。追込・後方一気は構造的に不利。
  • 直前開催（6/13）の決着は約57.8秒＝高速ダート寄り。

馬場状態

  • 馬場状態：当日発表前（ダート）。発走前にJRA公式で要確認。
  • 6月下旬の函館ダートは概ね締まった速い馬場。本予想は標準〜やや速い前提。
  • ダート1000mは天候の影響で時計が前後するが、前残り傾向自体は不変。

過去傾向

  • 函館ダ1000m・3歳未勝利は先行決着が基本。逃げ〜好位の馬が圧倒的有利。
  • 直前2開催の同条件で3着していた馬（スイーヴル・クインズスズラン）がそのまま今回も上位。
  • 後方一気は決まりにくい。

展開予測

  • メインシナリオ（ミドル〜やや速い・前残り）: 先行型が大半（マリブサーフ・アイビーミャーオ・ドリームガッサンが最前、スイーヴル・セラサイト・カレンココナが好位）。先行争いを生き抜き、かつ最も速い1000m時計を持つスイーヴルが中心。前に行けるマリブサーフが押し切るシナリオも。
  • サブシナリオ（さらに速い流れ）: 前が総崩れなら好位差しのクインズスズランや距離万能のセラサイトが浮上。ただし追込のアドバンスジェンヌが届くほどは緩まない見込み。

全頭評価（主要馬）

◎ スイーヴル（2番・2枠）47pt

前走: 函館ダ1000・3着（0.7差・時計58.5＝当該条件で当メンバー最速）
- 1.過去実績 4 / 2.レースレベル 3 / 3.血統 4 / 4.コース適性 5 / 5.輸送 3 / 6.休み明け 3 / 7.脚質展開 5 / 8.枠順 4 / 9.騎手 4 / 10.調教 4 / 11.馬体 3 / 12.厩舎 3 → 47pt（A）

良い点
- ダート1000mで最も実証された馬。小倉ダ1000で3着×2、函館ダ1000で3着と、当該条件で安定して3着圏。今メンバーで最速の1000m時計58.5（6/13・3着）。
- 先行できる脚質（4角位置0.34）で前残り強コースに最適。追い切りも当週トップ級（ウッド終い11.3）。
- 鞍上が岩田望来へ強化。父タワーオブロンドン＝スプリント血統。

不安材料
- 通算0勝（9戦）。3着は多いが勝ち切れない「ブライドスメイド」傾向。勝ち味に遅さがある。
- 複勝1.2-1.4＝過剰人気水準（D6で◎複勝は買わない）。

○ マリブサーフ（10番・7枠）44pt

前走: 新潟ダ1200・4着（0.8差）/ ダ1400で2着0.2・3着0.1の僅差好走
- 1.過去実績 4 / 2.レースレベル 3 / 3.血統 4 / 4.コース適性 3 / 5.輸送 3 / 6.休み明け 3 / 7.脚質展開 5 / 8.枠順 3 / 9.騎手 3 / 10.調教 2 / 11.馬体 3 / 12.厩舎 3 → 44pt（B+）

良い点
- 当該メンバーの実質地力No.1（hidden_potential 5pt★★）。ダ1400で勝ち馬と0.1〜0.2秒差の僅差2着・3着を続ける完成度。
- ハナを切れる前向きな脚質（4角0.26＝最前）。前残り強コースを先行力で押し切れる。父ロードカナロア＝スピード。

不安材料
- ダート1000mは未経験（1400→1000の大幅短縮）。流れが一気に速くなり、置かれるリスク。
- 最終追い切りがやや緩い（終い16.1）。

▲ クインズスズラン（6番・5枠）42pt

前走: 函館ダ1000・3着（0.6差・6/14・同舞台）
- 1.過去実績 3 / 2.レースレベル 3 / 3.血統 4 / 4.コース適性 4 / 5.輸送 3 / 6.休み明け 4 / 7.脚質展開 4 / 8.枠順 3 / 9.騎手 4 / 10.調教 2 / 11.馬体 3 / 12.厩舎 3 → 42pt（B）

良い点

△ チュラヴェール（9番・7枠）40pt

前走: 函館ダ1000・5着（1.3差・6/13＝◎スイーヴルと同レースで0.6秒劣後）
- 福島ダ1150で2着・3着の安定感。丹内祐次が6戦継続。追い切りは強め（終い11.7）。
- ただし同じ函館ダ1000でスイーヴルに直接0.6秒負けており、現状は上位2頭の下。3人気は買いかぶり気味。

△ ドリームガッサン（7番・6枠）39pt

前走: 新潟ダ1200・6着 / 福島ダ1150で2着0.7
- 最前で運べる脚質（4角0.29）。鞍上小林美駒は函館ダ短距離で回収率128%の妙味（+1）。
- 小柄（434kg）で相手強化時に甘くなる面。

☆ セラサイト（12番・8枠）38pt

前走: 函館ダ1000・4着（0.8差・6/14）

- 当該舞台で掲示板は確保するが勝ち負けまでは半信半疑。3列目押さえ。
