項目 内容

予想信頼度

TIER2 中信頼（最高44pt / 1-2位点差1pt / 14頭）

  • 平場かつ「超荒れる」傾向レースのためTIER1は出さない
  • 上位人気が5倍前後に複数並ぶ拮抗構成。軸の絶対的信頼は置けない
  • 過去同レース（京都ダ1400化後の2023-2025）は毎年6〜8番人気が3着内に絡む

コース特性（京都ダート1400m）

  • スタート後すぐ3コーナーに入る形で、序盤からポジション争いが激しい
  • 直線は約330m。先行有利の傾向が強い
  • 京都ダ1400・2勝クラス（2023-2026）の3着内159サンプルでは、4コーナー1〜3番手＝67回 / 4〜7番手＝49回 / 8番手以降＝43回。前で運べる馬が有利、極端な後方一気は割引
  • 枠順は内〜中枠がロスなく立ち回れる。外枠は最初のコーナーまでに脚を使わされやすい

馬場状態

  • 馬場状態：未発表（当日朝の発表待ち）
  • 当日の天候・馬場はDB（fact_race）に未反映。良想定で評価しつつ、道悪化した場合は前で運べる先行馬の比重を上げる
  • 取得時刻：オッズは2026/05/23 03:25時点（netkeiba隠れAPI）

過去傾向（オーストラリアトロフィー＝京都ダ1400化後3年）

  • このレースは2022年以前は芝・別距離（1200/1600/1800m）で実施されており、2023年から京都ダート1400mの2勝クラスとして固定。傾向集計は2023-2025の3年が母集団
  • 2025年: 1着プロミシングスター(2人気) / 2着フェンダー(3人気) / 3着モズナイスバディー(6人気)
  • 2024年: 1着パーティーベル(4人気) / 2着カッティングジェム(8人気) / 3着シゲルソロソロ(1人気)
  • 2023年: 1着ヴアーサ(1人気) / 2着スキピオ(3人気) / 3着スマートセプター(7人気)
  • 3年とも6〜8番人気が1頭は3着内に突っ込んでいる。上位人気だけで決まらない
  • 前年同レース出走馬: ⑦マイティマイティー（2025年5着・1番人気で凡走）

展開予測

  • メインシナリオ（ミドルペース）: 突出した逃げ馬が不在で、前へ行きたい馬が複数。⑥ダノンキラウェア・④ヨリノレジェンド・②リアルスターあたりが好位を取りに行く形。先行有利のコース形態を踏まえ、3〜6番手で脚をためられる馬が恵まれる
  • サブシナリオ（ややハイ寄り）: 外枠の⑬⑭が出していくとペースが上がり、中団から差してくる⑧ユウトザレン・⑩ハリウッドブルースに展開が向く。⑭ヴリトラハンは過去ハイペースで複勝率57%・ミドルで0%と、流れが速くなる方が良い

1.過去実績 4/5 芝で甲東特別3着・京都芝1400で3着など、勝ち馬と0.1秒差クラスの僅差好走を複数

3.血統 4/5 父ロードカナロアはダート短距離も走る。母父Sligo Bayでパワー補完
4.コース適性 3/5 京都ダ1400は経験薄いが、前走阪神ダ1400・吹田特別で3着と好走。ダート短距離に対応済み
5.輸送適性 3/5 関西圏中心で輸送負担は小さい
6.休み明け 4/5 通常ローテで安定。間隔が空かない方が良いタイプ（得意条件「通常間隔」で掲示板率100%）

良い点
- 着順に表れない実力（僅差負け・上がり上位）の蓄積が多く、地力は出走馬中でも上位
- 前走で阪神ダ1400・吹田特別を経験し、ダート短距離への適性を実証した直後
- 西村淳也との主戦コンビ継続で、脚質・気性を把握した騎乗が見込める

不安材料
- 京都ダ1400そのものの経験が薄く、コース実績は前走の阪神ダ1400から類推
- 持ち時計面で抜けた存在ではなく、展開ひとつで取りこぼす可能性

期待点
- 前で運べる脚質×先行有利コースの組み合わせ。展開利を最も受けやすい1頭

買い目提案

レース性質「超荒れる」＋平場のため、上位2頭軸を基本にしつつ3列目を広く取る方針。buying-rules.md テンプレC（二段構え）をベースに、3連複の相手を中穴まで広げる。1レース500円。

複勝（1点）

複勝 単独
  - ⑥ ダノンキラウェア 100円
  → ⑥が3着以内で的中

※◎⑥の複勝想定は1.8-2.1倍。最低倍率1.5倍は上回るため購入可。

3連複 軸2頭ながし（◎○軸・本線）

3連複 軸2頭ながし
  - 軸: ⑥ ⑥は◎ / ④は○ → 軸 ⑥④
  - 相手: ⑧ ⑭ ⑬ ②
  - 点数: 相手4頭ぶん = 4点
  - 1点 × 50円 = 200円
  - 検算: 4点 × 50円 = 200円
  → ⑥と④の2頭 + 相手1頭の計3頭が3着以内で的中

3連複 軸1頭ながし（○軸・◎不在保険）

3連複 軸1頭ながし
  - 軸: ④（○）
  - 相手: ⑧ ⑭ ⑬
  - 点数: 相手3頭から2頭選ぶ組合せ = 3C2 = 3点
  - 1点 × 50円 = 150円
  - 検算: 3点 × 50円 = 150円
  → ◎⑥が飛んでも、④と相手2頭が3着以内なら的中（◎不在保険）

合計

複勝          100円
3連複(◎○軸)  200円
3連複(○軸保険) 150円
─────────────────
合計          450円

※500円基本に対し450円。50円単位購入はネット投票で要確認（不可なら3連複本線を100円×4＝400円に増額し、複勝100円と合わせ500円に調整、保険分はそのとき割愛）。

買い目の意図

  • ◎○の信頼度は高いが「超荒れる」レースなので◎1頭軸に依存しない。3連複の軸を◎○の2頭に置き、3列目に中穴⑬⑬・⑧・②まで広げる
  • ▲⑭・☆⑬は相手に必ず組み込み、拾った馬を買い目から外さない原則を満たす
  • ○④軸の◎不在保険を独立して1点以上確保（TIER2かつ超荒れるレースのため）
  • ×⑫は買い目に含めない（過剰人気の消し馬）。注⑦⑩は予算500円の枠内で3連複本線の相手に⑬まで含めることで実質カバー範囲に入れているが、独立した保険までは予算上組まない（妙味と予算のバランス）
