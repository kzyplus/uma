項目 内容

予想信頼度

TIER2 中信頼（最高47pt / 1-2位点差1pt / 15頭）
- 平場のためTIER1は付けない方針
- 過去同条件（新潟ダ1800・2021-2025）: 1〜3番人気の3着内占有率52.6%、8番人気以下の3着内出現率48.0% ＝ 中程度（やや荒れ）

コース特性

  • 新潟ダート1800mは左回り。スタート後の最初のコーナーまで距離があり、序盤の枠の不利は出にくい
  • 直線は約354m。ダートとしては平坦寄りで、極端な瞬発力勝負にはなりにくい持続力型
  • 枠順傾向（2021-2025・3着内集計）: 外枠（5〜8枠）が好走多数。最内1〜2枠は明確に手薄。今回◎⑮（8枠）・○④（3枠）は枠で割引不要、△①（1枠最内）はマイナス
  • 脚質はDB上の脚質コードが未整備で集計不能。コース形態から「先行〜好位差し」がやや有利と一般傾向で補完

馬場状態

  • 馬場状態：当日発表前（未取得）
  • 想定: 5月下旬の新潟開催。当日朝のJRA公式発表で確定する
  • 道悪（稍重以上）になった場合、ハギノコラソンは道悪巧者でむしろプラス転化（過去・道悪条件の好走率が高い）

過去傾向（新潟ダ1800m・過去5年）

  • 1〜3番人気の3着内占有率52.6% ＝ 上位人気は信頼しつつ、半分は人気薄が3着内に食い込む
  • 8番人気以下の3着内出現率48.0% ＝ 1レースに1頭程度は中穴〜大穴が突っ込む
  • 外枠有利・内枠不利が明確
  • 本レースは特別名のない平場のため「前年結果」「リピーター」概念は適用外

展開予測

  • メインシナリオ（ミドルペース）: 明確な逃げ宣言馬が不在。マタミカンサン①・テルケンユミクイン⑤あたりがハナを主張する可能性はあるが、隊列が落ち着けば淡々としたミドル。差し・好位勢に展開が向く。恩恵: 好位から運べるハギノコラソン⑮・キタノマーシャル④。前走当コースを好位2-3着したオブシディアーナ⑧も持続力勝負はプラス
  • サブシナリオ（ややハイ）: テルケンユミクイン⑤が前走中山ダで先行して失速しており、無理に行くと前崩れ。その場合は後方一気が届きやすく、追い込み脚質のムーランブルー⑫・ブーケドグレイス⑭の3着差し込みに警戒

買い目提案（テンプレC：二段構え・500円基本）

レース荒れ度は中程度（やや荒れ）。◎の信頼度は高いが「2-3着が多く勝ち切れない」脚質のため、
◎1頭軸偏重を避け ◎○の2頭を軸の柱 に据える。

馬連 軸1頭ながし（◎軸）

  • 馬券種: 馬連
  • 買い方の種類: 軸1頭ながし
  • 軸馬: ⑮ハギノコラソン
  • 相手馬: ④⑧①
  • 点数: 3点
  • 1点あたり金額: 100円
  • 合計金額: 3 × 100 = 300円
  • 想定的中条件: ⑮が1〜2着 ＋ 相手（④⑧①）のうち1頭が1〜2着

3連複 軸2頭ながし（◎○2頭軸）

  • 馬券種: 3連複
  • 買い方の種類: 軸2頭ながし
  • 軸馬: ⑮ハギノコラソン・④キタノマーシャル
  • 相手馬: ①⑧③⑥
  • 点数: 4点（相手4頭＝4点）
  • 1点あたり金額: 50円
  • 合計金額: 4 × 50 = 200円
  • 想定的中条件: ⑮④の2頭 ＋ 相手（①⑧③⑥）のうち1頭が1〜3着

合計

300円 + 200円 = 500円（テンプレC・基本予算）

検算

馬連 軸1頭ながし: 軸⑮ - 相手④⑧① = 3点 × 100円 = 300円
3連複 軸2頭ながし: 軸⑮④ - 相手①⑧③⑥ = 4点 × 50円 = 200円
合計: 300 + 200 = 500円

補足（買い目に含めた「拾った馬」の確認）

  • ☆⑥ツァガーンサル → 3連複の相手4頭に含めた（穴馬スクリーニング反映済）
  • 注⑭ブーケドグレイス・⑫ムーランブルー（地力★★★馬）→ 500円基本では枠の都合で本買い目に直接は入らないが、地力★★★馬の3連系3列目保険ルールに照らし、予算を1,000円に拡張する場合は下記「拡張オプション」で必ず1点保険を組むこと。500円縛りでは⑥を相手に入れることで「地力評価で拾った穴」の代表として網をカバー

拡張オプション（1日1Rのみに賭ける場合・テンプレD / 1,000円）

複勝 単独: ⑮ハギノコラソン 200円
馬連 軸1頭ながし: 軸⑮ - 相手④⑧①③ = 4点 × 100円 = 400円
3連複 軸2頭ながし: 軸⑮④ - 相手①⑧③⑥⑭⑫ = 6点 × ... → 50円換算で300円(6点) + 端数調整
  → 実際は 軸⑮④-相手①⑧③⑥⑭⑫ 6点 × 50円 = 300円 + 複勝100円増額で合計1,000円
合計: 複勝300円 + 馬連400円 + 3連複300円 = 1,000円

→ 拡張時は3連複の相手に⑭⑫を加え、地力★★★馬の3列目保険を達成する。
