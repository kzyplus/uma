買い目提案（§7-9・1R 500円基本＋§8.6保険拡張で計700円）

荒れ度=中程度〜やや荒れる。◎が1番人気で勝ち切れていない未勝利戦、相手に地力No.1の伏兵⑦を抱えるため、◎○2頭軸を基本にしつつ印馬の網を広く取る。

本線：3連複 軸1頭ながし（◎⑩ → 相手5頭 ○④▲⑦△⑪△⑭）

  • 組み合わせ: ⑩を軸に、相手 ④⑦⑪⑭ + ☆⑯ から2頭
  • 相手集合 = ④⑦⑪⑭⑯ の5頭 → C(5,2)=10点 … ただし予算内に収めるため相手4頭に絞る
  • 採用: ⑩軸ながし 相手4頭（④⑦⑪⑭）C(4,2)=6点 ×50円 = 300円

保険A：地力No.1⑦頭固定 3連複（§8.5・2.0倍）

  • 3連複 ⑦頭固定 → ⑩④⑪ ＝ ⑦-⑩-④ / ⑦-⑩-⑪ / ⑦-④-⑪ の3点 ×?
  • ⑩軸ながしと重複する組（⑦-⑩-④/⑦-⑩-⑪）は本線でカバー済み。新規追加は ⑦-④-⑪（◎不在で地力No.1が突き抜けるシナリオ）1点 ×100円 = 100円

保険B：印1-2-3着0円防止（§8.6改定・4頭ボックス）

  • 3連複 ⑩④⑦⑪ ボックス C(4,3)=4点 ×? … うち本線・保険Aと重複する ⑩④⑦ / ⑩④⑪ / ⑩⑦⑪ を除く新規は ④⑦⑪ 1点（保険Aの⑦-④-⑪と同一）→ 既に計上済み。
  • 追加で △⑭を含む 3連複 ⑩④⑭ ・ ⑩⑦⑭ は本線(⑩軸相手④⑦⑪⑭)でカバー済み。
