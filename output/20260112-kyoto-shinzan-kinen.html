惜しかった点：3番サンダーストラックを評価4位・43点としていたが、買い目に含めなかった。穴馬スクリーニングで「★★★☆☆」評価だったにも関わらず、買い目提案では除外していた。

反省点

❶ 本命の過大評価：13番モノポリオ（49点）を本命としたが5着敗退。重賞実績を重視しすぎた。
❷ 穴馬の買い目除外：評価4位の3番サンダーストラックを穴馬候補としながら、買い目に含めなかった。
❸ 未勝利馬の過小評価：3番は「新馬戦未出走」をマイナス評価したが、血統（ロードカナロア）・調教（S評価）が的中。経験不足を過度に懸念した。
❹ 買い目構成の硬直性：本命軸一辺倒で、対抗1番を絡めた買い目が足りなかった。1-3の組み合わせを本線に含めるべきだった。

なぜ本命が外れたのか？

❶ 重賞実績の過大評価
モノポリオはアイビーS（L）3着の実績を評価したが、同レースは8頭立ての小頭数で、1着馬（ウエストナウ）が今回不在。前走の相手レベルが低かった可能性を見逃した。フェアリーS教訓を適用したが、小頭数レースの実績は割引評価すべきだった。
❷ 未勝利馬の潜在能力を軽視
サンダーストラック（3番）は「新馬戦未出走」をリスク視したが、父ロードカナロア×京都芝1600mの血統的優位性、追い切りS評価（5F69.3秒-上がり11.4秒）を過小評価。血統・調教が揃えば、未勝利でも上位争い可能であることを実証した。
❸ 外国人騎手の技術力を見誤った
T.ハマーハンセン騎手を「外国人騎手の技術力」として3点評価としたが、実際にはJRA重賞初制覇を果たした。外国人騎手の技術力は、日本のトップ騎手と同等かそれ以上であり、騎手項目の評価基準を見直す必要あり。
❹ 枠順の影響を過小評価
サンダーストラックは2枠3番の好枠。京都芝1600m外回りは内枠有利の傾向があるが、評価時には枠順3点と平凡な評価に留めた。実際には内枠から理想的な競馬をして勝利。枠順項目の配点を見直すべき。

今後への教訓

教訓①：小頭数レースの実績は割引評価 → 8頭以下の重賞実績は、実績点から-1点する。相手関係が薄い可能性を考慮。
教訓②：血統×調教×騎手が揃えば、未勝利でも警戒 → 血統4点以上 + 調教5点 + 騎手4点以上の馬は、実績不足でも評価を+2点上乗せする。
教訓③：穴馬候補は必ず買い目に含める → 穴馬スクリーニングで★★★以上の馬は、ワイド・3連複の押さえに必ず含める。評価だけで終わらせない。
教訓④：外国人騎手の評価基準を見直す → T.ハマー、M.デムーロ、C.ルメールなど、外国人騎手の騎手項目は4〜5点を基本とする。技術力は国内トップ級と同等以上。
