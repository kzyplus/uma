コース特性（函館ダ1700m）

  • 右・小回り・平坦。直線が短く（ダート260m級）先行有利・前残りの強いコース。
  • 過去5年勝ち馬は4角1〜3番手が大半。netkeiba集計（直近1年・4角別複勝率）は逃げ41%/先行57%/後方5〜6%＝前で運べる馬が圧倒的に有利。
  • ダートは外枠やや有利（砂被り回避）だが小回りで内の先行も生きる。極端な枠補正はしない。

馬場状態・馬場差

② 馬連（主役・4点 400円）

印上位4頭（②⑬⑪⑫）の組み合わせを軸②から流す＋対抗同士:
- 馬連 ②-⑬ / ②-⑪ / ②-⑫ … 各100円（◎軸流し3点）
- 馬連 ⑬-⑪ … 100円（◎不在保険＝○▲）
- 合計 4点×100円 = 400円

③ 3連複 2軸ながし（◎②＋○⑬ → 相手⑪⑫⑨⑥）（補助・6点 300円）

  • 3連複 ②⑬ - ⑪⑫⑨⑥（軸2頭ながし） … C(4,1)=4通り…ではなく ②⑬固定で相手4頭から1頭 = 4点
  • 組み合わせ: ②⑬⑪ / ②⑬⑫ / ②⑬⑨ / ②⑬⑥ … 4点
  • 3連複 印1-2-3着保険（◎○▲△ボックス・§8.6）: ②⑬⑪⑫ のうち未カバー分 → ②⑪⑫ / ⑬⑪⑫ … 2点
  • 合計 6点×50円 = 300円

④ 3連複 3列目で穴をケア（注①③⑦）（保険・2点 100円）

  • 3連複 ②⑬ - ①③⑦（3列目に注の穴を1点ずつ）から2点に絞る: ②⑬① / ②⑬③ … 各50円

検算

  • 複勝 200 ＋ 馬連 4×100=400 ＋ 3連複(2軸+保険) 6×50=300 ＋ 3連複(穴) 2×50=100 = 1,000円
