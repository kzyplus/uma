予想信頼度

TIER2 中信頼（最高47pt / 1-2位点差2pt / 15頭）

  • 平場のため情報量が限られTIER1は出さない方針。
  • 12項目1位の⑩マディソンガールが地力・当該コース実績とも断トツだが、休み明け＋追い切り終い甘めの割引材料あり。○⑥との点差は2ptで上位2頭は信頼可。3列目以下は混戦。

コース特性（京都芝1800m 外回り）

  • 外回りコース。スタート後の最初のコーナーまで距離があり、序盤の位置取り争いは比較的落ち着きやすい。
  • 直線は約404m＋緩やかな坂。長い直線で末脚を問われるが、ペースが上がりにくく前残りも多い。
  • 直近データ（2024-2026年・3着内204サンプル）: 4コーナーを8番手以内で回った馬が3着内の約90%を占める＝前〜好位有利、後方一気は決まりにくい。
  • 枠順: 内外ほぼフラット（5枠・7枠・8枠がやや good）。極端な枠の有利不利は小さい。
  • 人気別では1-3番人気が3着内の60%＝平場としてはやや堅め。

馬場状態

  • 馬場状態：当日未確定（良想定で進行）
  • 5/23当日のJRA公式発表をDBに未反映。前日時点で雨予報の強い情報はなく、芝は良〜稍重想定。
  • 道悪に大きく振れた場合は、差し・追い込み勢の台頭余地が出るため展開予想を見直す前提。

過去傾向（京都芝1800m外・1勝クラス）

  • 前残り傾向が明確で、逃げ・先行〜好位差しが中心。
  • 上がり最速を出しても4コーナーで後方すぎると届かないケースが多い。
  • 1勝クラスは「同条件（場×距離×芝）で複勝圏」の実績がそのまま強い指標になる。

展開予測

  • メインシナリオ（ミドル〜ややスロー）: 15頭立てだが明確な逃げ馬不在で、隊列は落ち着きやすい。先行できる⑨ハルフロンティア・好位を取れる⑪ジェゼロ・⑩マディソンガールが恩恵。差し・追い込みの⑥ピエタンツァ・③ウインラウダ・⑤メイショウタクボクは展開待ちで、上がり勝負を望む。
  • サブシナリオ（道悪・前崩れ）: 馬場が渋って前が止まれば、上がり最速級の脚を持つ⑥ピエタンツァ・③ウインラウダ・⑤メイショウタクボクに出番。⑩は好位差しで対応可だが、後方一気タイプは恩恵が大きくなる。

買い目提案

方針: TIER2・上位2頭（◎⑩・○⑥）は信頼できるが、◎は休み明け＋追い切り終い甘めの割引があり1頭軸偏重は避ける。◎○2頭軸を基本に、◎不在・○不在の保険も残す。複勝は◎が1.1倍前後で妙味がないため使わない。1レース500円・テンプレA（本命堅い）をベースに馬連＋3連複でミックス。

馬連 軸1頭ながし（◎軸）

  • 馬券種: 馬連
  • 買い方の種類: 軸1頭ながし
  • 軸馬: ⑩
  • 相手馬: ⑥⑮⑨
  • 点数: 3点
  • 1点あたり金額: 100円
  • 合計金額: 3点 × 100円 = 300円
  • 想定的中条件: ⑩が1〜2着 + 相手（⑥⑮⑨）のうち1頭が1〜2着

3連複 軸2頭ながし（◎○軸）

  • 馬券種: 3連複
  • 買い方の種類: 軸2頭ながし
  • 軸馬: ⑩・⑥
  • 相手馬: ⑮⑨③⑪
  • 点数: 4点（=相手4頭ぶん）
  • 1点あたり金額: 50円
  • 合計金額: 4点 × 50円 = 200円
  • 想定的中条件: ⑩と⑥の両方 + 相手（⑮⑨③⑪）のうち1頭が1〜3着

合計

300円 + 200円 = 500円

◎不在保険・○不在保険の確認: 3連複の軸2頭ながしは「⑩⑥のどちらか1頭が飛ぶと不的中」になるため、500円枠内で別途1点保険を組めない構成上の弱点がある。これを補うため、1日1Rに集中する場合はテンプレD-1（1,000円拡張）を推奨。下記に拡張案を併記する。

【1日1Rのみに集中する場合】テンプレD-1拡張 — 1,000円

  • 馬連 軸1頭ながし: 軸⑩ → 相手⑥⑮⑨ … 3点 × 100円 = 300円
  • 3連複 軸1頭ながし（◎軸）: 軸⑩ → 相手⑥⑮⑨③⑪ … 10点（=5C2）× 50円 = 500円
  • 3連複 フォーメーション（◎不在・○不在保険）: ⑥-⑮-⑨ / ⑥-⑮-③ / ⑨-⑮-③（▲△△の連下同士組合せ＋○軸）… 3点固定にならないよう、⑥⑮⑨③の4頭から ◎を含まない組合せを 1列目⑥, 2列目⑮⑨, 3列目⑨③ で 2点 × 100円 = 200円
  • 合計: 300 + 500 + 200 = 1,000円
  • ◎不在でも ⑥⑮⑨/⑥⑮③ で的中可。連下同士（▲⑮-△⑨, ▲⑮-△③, △⑨-△③）も3連複側で全網羅。

⑤メイショウタクボク（大穴）の3列目1点保険は、上記500円基本案では予算超過のため見送り。1,000円拡張を採る場合は、3連複軸1頭ながしの相手に⑤を1頭加える（6頭で15点×50円=750円）選択も可。ただし的中効率を優先し、本案では相手5頭に絞る。
