コース特性（阪神ダート2000m）

  • スタンド前直線入口からのスタートで最初のコーナーまで距離があり、序盤の枠順不利は小さい。
  • 1周強を回るためペースは緩みやすく、追走力＋折り合い＋一定の上がり脚が要るタフな舞台。
  • 距離2000mは3歳未勝利では長め。1800〜1900mで安定して掲示板に載れる持久力が問われる。
  • 紛れは出やすく（超荒れる条件）、人気薄の地力上位が突っ込む余地がある。

買い目提案（テンプレC＋§8.13 券種使い分け）

方針: 超荒れる条件だが地力No.1の◎⑥を堅実軸に据えられる。§8.13に従い複勝・馬連を主役、3連複は全体の3割以内（4頭固定でなく軸2頭ながし2点）。妙味中穴△⑭（9人気）は単複でなく複勝＋馬連で馬を買う。予算は§8.6印保険を含め800円に拡張（1日上限内）。

① 複勝（2点）= 主役

  • ◎⑥ピエナオルカ 200円
  • △⑭クリノリキオー（9人気・妙味中穴） 100円
  • 小計: 300円

② 馬連 ◎軸1頭ながし（3点）= 主役

  • ⑥-⑩、⑥-⑯、⑥-⑭ 各100円×3
  • 小計: 300円

③ 3連複 ◎○軸2頭ながし（2点）= 補助（全体の25%）

  • ⑥-⑩ 軸 → ⑯・⑭ 各100円×2
  • 展開: 6-10-16（◎○▲＝§8.6 印1-2-3着保険を内包）／6-10-14（◎○△）各100円
  • 小計: 200円

検算

複勝   ⑥200 + ⑭100              = 300円
馬連   ⑥-⑩,⑥-⑯,⑥-⑭ 100×3       = 300円
3連複  6-10-16, 6-10-14  100×2     = 200円
------------------------------------------------
合計                              = 800円
3連複比率 200/800 = 25%（3割以内 ✓）
印保険（§8.6）: 6-10-16 ＝ ◎○▲ を内包 ✓
3頭固定フォーメーションなし（軸2頭ながし ✓）
ワイドに本命を入れていない ✓ / 単勝全頭買いなし ✓

的中イメージ

  • ◎⑥が3着内 → 複勝的中。◎⑥が連対し相手が⑩⑯⑭のいずれか → 馬連的中。
  • ⑥⑩が上位で⑯か⑭が絡む → 3連複的中（高配当狙い）。
  • 印1-2-3着（⑥⑩⑯ or ⑥⑩⑭）が揃えば馬連＋3連複のダブル的中。
