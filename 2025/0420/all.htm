2025年 皐月賞 全頭評価

※現時点の想定騎手・戦績・血統・適性に基づく予想印（◎〇▲△☆）を記載しています。

☆ エリキング （川田将雅）

  • 血統: 父キズナ×母ヤングスター。母は豪オークス馬で、欧州的なスタミナ血統との融合型。
  • 馬体: 470〜480kg台。ゆったりとした体型で成長力も見込め、トモの発達も良好。
  • レース内容: 新馬→野路菊S→京都2歳Sと無傷の3連勝。野路菊Sではクロワデュノールに先着歴あり。
  • 騎手: 川田将雅が手塩にかける有力馬で、クラシック制覇への本気度が高い。
  • 評価: 荒削りながら完成度と成長性のバランス良く、無敗の勢いを侮れない。対抗格。

☆ ファウストラーゼン （杉原誠人）

  • 血統: 父モズアスコット×母ペイシャフェリス。父は芝ダートG1馬で、本馬はスタミナ型の配合。持続力とタフさが武器。
  • 馬体: 480kg台でパワフルな体つき。使われながら完成度が増しており、調整も順調。
  • レース内容: 弥生賞では向正面からまくって押し切り勝ち。ホープフルSでも好走。展開に左右されにくい。
  • 騎手: 杉原誠人が続投。弥生賞で重賞初制覇をもたらし、クラシック初挑戦にも気合十分。
  • 評価: 地味だが崩れない堅実型。まくり脚質がハマればG1でも好勝負可能な穴候補。

☆ クロワデュノール （北村友一）

  • 血統: 父キタサンブラック×母ライジングクロスという良血で、母は英オークス・愛オークス2着。父産駒には2023年皐月賞馬ソールオリエンスも。
  • 馬体: 馬体重は480kg→504kg→496kgと推移。1週前追い切りでCWラスト1F10.9秒を記録し、仕上がりに不安なし。
  • レース内容: 無敗でホープフルS制覇（上がり3F33.9秒）、東スポ杯2歳Sも33.4秒の末脚で快勝。全戦で最速級の上がりを記録。
  • 騎手: 北村友一騎手が新馬戦から継続騎乗。調教師も「仕上がりは万全」と明言しており、勝負気配高。
  • 評価: 血統・馬体・レース内容・騎手すべてにおいて隙がなく、完成度で頭ひとつ抜けている本命候補。

○ サトノシャイニング （西村淳也）

  • 血統: 父キズナ×母スウィーティーガール。キズナ産駒らしく、切れとスタミナを兼備。母系は地味だが、配合バランスが良く、クラシック向き。
  • 馬体: 馬体重は500kg前後で安定。胴伸びがあり体の使い方が柔らかい。杉山調教師も「バランスが良い」と高評価。
  • レース内容: きさらぎ賞では稍重馬場を先行して押し切り、3馬身差快勝。東スポ杯では落鉄がありながらクロワデュノールと0.1秒差の2着。
  • 騎手: 若手の西村淳也が継続騎乗。G1経験は浅いが、陣営の信頼厚く、今後の飛躍が期待される。
  • 評価: 能力的にはトップクラス。ただし小回りでの器用さや中山適性では若干未知数。地力と安定感に期待して対抗格に。

▲ ジョバンニ （松山弘平）

  • 血統: 父エピファネイア×母ベアフットレディ。母は英G1馬で欧州マイル系の良血。エピファネイア産駒として中山適性・クラシック適性十分。
  • 馬体: 馬体重500kg前後、体幹強く好馬体。トモの筋肉も発達し、急坂対応の下地あり。
  • レース内容: ホープフルSで2着と中山2000mの実績あり。前走若葉Sも好内容の完勝で重賞級と互角の力。
  • 騎手: 松山弘平が継続騎乗。陣営の信頼も厚く、G1での経験豊富。厩舎は同時出走馬サトノシャイニングとも好調。
  • 評価: 実績・安定感とも申し分なく、中山適性も証明済み。人気以上の評価必要。

　 マスカレードボール （横山武史）

  • 血統: 父ドゥラメンテ×母マスクオフ。父は皐月賞馬で産駒にもジオグリフなどクラシック実績あり。母系は社台系の良血。
  • 馬体: 500kg前後の雄大な馬体。緩さ残るも春に向けて絞れており、坂路でも良い動き。
  • レース内容: 共同通信杯で上がり最速33.2秒を記録し差し切り勝ち。瞬発力勝負に強み。
  • 騎手: 横山武史が継続騎乗。重賞実績豊富で、手塚厩舎ともクラシック好相性。
  • 評価: 良馬場・東京向きタイプ。中山替わりと多頭数が課題となり、評価は据え置き。

△ ミュージアムマイル （モレイラ）

  • 血統: 父リオンディーズ×母ミュージアムヒル。母はハーツクライ産駒で中長距離向き、父の産駒は中山巧者が多く好相性。
  • 馬体: 470kg台でシャープなシルエット。体の伸びがあり、直線での伸びに活きる形。
  • レース内容: 若葉Sで2着。上がり最速を記録し、展開に左右されず末脚堅実。
  • 騎手: モレイラ騎手が引き続き騎乗予定。天才ジョッキーとして人気以上の仕掛けに期待。
  • 評価: 末脚堅実で展開次第では浮上可能。善戦タイプとして穴候補の1頭。

△ ヴィンセンシオ （ルメール）

  • 血統: 父リアルスティール×母シーリア。母は超良血で祖母は名牝アドマイヤグルーヴ、牝系に名馬が多数。
  • 馬体: 470kg台で均整の取れた体型。ディープ系らしい柔らかさとキレを感じる。
  • レース内容: スプリングSで最速の上がりで鋭く伸びて2着。距離延長も問題なく、むしろ歓迎の可能性。
  • 騎手: ルメールが騎乗。クラシックでも抜群の安定感を誇る。
  • 評価: 距離が延びて良さが出るタイプ。大舞台での騎手効果も加味して押さえ候補。

　 ジュタ （坂井瑠星）

  • 血統: 父ドゥラメンテ×母シャンパンエニワン。母系は米国スピード血統で、距離に課題が残る。
  • 馬体: 470kg前後、筋肉量豊富でマイラー寄りの印象。
  • レース内容: スプリングSで3着と健闘も、重賞ではあと一歩足りない印象。
  • 騎手: 坂井瑠星が引き続き騎乗予定。積極策で乗るタイプ。
  • 評価: 距離延長と相手強化が不安材料。抑えまで。

　 アスクシュタイン （調整中）

  • 血統: 父ドゥラメンテ×母ヴィクトリアズワイルドキャット。母は米重賞勝ち馬で、芝2000m前後に適性がある配合。
  • 馬体: 490kg前後、骨太で力強さのある体つき。調整過程も順調で完成度は高い。
  • レース内容: 近走は条件戦を安定して勝ち上がり。重賞経験はないが、伸びしろは大。

  • 評価: 素質馬だが実績不足。いきなりG1では評価は控えめ。

　 カラマティアノス （戸崎圭太）

  • 血統: 父レイデオロ×母ダンサール（母父ハーツクライ）。サンデー×キングカメハメハ×ハーツの主流配合。
  • 馬体: 470kg前後、バランス型。成長途上だが全体的にまとまりあり。
  • レース内容: 前走は自己条件で完勝。中山での実績がなく、重賞実績も不十分。
  • 騎手: 戸崎圭太が騎乗予定。重賞での経験豊富で、好位差しに持ち込めればチャンスも。
  • 評価: 素質は認めるが現時点では様子見。人気薄での激走に注意。

　 キングスコール （藤岡佑介）

  • 血統: 父ドゥラメンテ×母レインオンザデューン（母父Frankel）。欧州名門牝系にドゥラメンテを掛け合わせた長距離志向の配合。

  • レース内容: 前走条件戦勝利もメンバーレベルはやや低め。時計面での裏付けが乏しい。
  • 騎手: 藤岡佑介が引き続き騎乗。重賞実績あり安定感はあるが、皐月賞でのインパクトは薄い。
  • 評価: 潜在能力は感じるが、今回のメンバーでは力不足感あり。経験積んでからの巻き返しに期待。

　 ニシノエージェント （津村明秀）

  • 騎手: 津村明秀が騎乗。堅実だがG1での実績は少なく、今回は試金石。

　 ピコチャンブラック （石橋脩）

  • 血統: 父キタサンブラック×母トランプクイーン。父譲りの持久力型。ネオユニヴァースの底力も期待。

  • 騎手: 石橋脩が手綱。先行力を活かしたレースが鍵。
  • 評価: スタミナ戦なら浮上余地あり。馬場悪化など助けあれば穴候補。
